Gets next window in the Z order, skipping invisible and other windows that probably are not in the Windows taskbar.
Returns default(Wnd) if there are no such windows.

Parameters

w (Optional)
Type: AuWnd
Start from this window. If default(Wnd), starts from the top of the Z order.
allDesktops (Optional)
Type: SystemBoolean
On Windows 10 include windows on all virtual desktops. On Windows 8 include Windows Store apps (only if this process has UAC integrity level uiAccess).
skipMinimized (Optional)
Type: SystemBoolean
Skip minimized windows.
retryFromTop (Optional)
Type: SystemBoolean
If w is not default(Wnd) and there are no matching windows after it, retry from the top of the Z order. Then can return w.

Return Value

Type: Wnd
Remarks
Uses Wnd.GetWnd.IsMainWindow.
This function is quite slow. Does not match the order of buttons in the Windows taskbar.
