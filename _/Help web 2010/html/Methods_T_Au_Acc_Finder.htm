Finder Methods

Au Class Library
Methods
  NameDescription
Find(Acc)
Finds accessible object (AO) in another AO, like Acc.Find.
Returns true if found. The Acc.Finder.Result property will be the found AO.
Find(Wnd)
Finds accessible object (AO) in window w, like Acc.Find.
Returns true if found. The Acc.Finder.Result property will be the found AO.
Find(Wnd, WndChildFinder)
Finds accessible object (AO) in the specified control of window w.
Returns true if found. The Acc.Finder.Result property will be the found AO.
Wait(Double, Acc)
Finds accessible object (AO) in another AO.
The same as Acc.Finder.Find, but waits until the AO is found or the given time expires.
Wait(Double, Wnd)
Finds accessible object (AO) in window w.
The same as Acc.Finder.Find, but waits until the AO is found or the given time expires.
