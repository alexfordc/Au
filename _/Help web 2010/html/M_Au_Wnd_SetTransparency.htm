Parameters

allowTransparency
Type: SystemBoolean
Set or remove WS_EX_LAYERED style that is required for transparency. If false, other parameters are not used.
opacity (Optional)
Type: SystemNullableDouble
Opacity from 0.0 (completely transparent) to 1.0 (opaque). If null, sets default value (opaque).
colorRGB (Optional)
Type: SystemNullableColorInt
Make pixels painted with this color completely transparent. If null, sets default value (no transparent color). The alpha byte is not used.
