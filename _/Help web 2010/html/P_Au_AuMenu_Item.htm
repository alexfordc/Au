AuMenuItem Property

Au Class Library
Adds new item as System.Windows.Forms.ToolStripMenuItem.
Sets its text, icon and Click event handler. Other properties can be specified later. See example.
Code m.Add("text", o => Print(o)); is the same as m["text"] = o => Print(o); .

Parameters

text
Type: SystemString
Text. If contains a tab character, like "Open\tCtrl+O", displays text after it as shortcut keys (right-aligned).
icon (Optional)
Type: SystemObject
Can be:
string - path of .ico or any other file or folder or non-file object. See Icon_.GetFileIcon. If not full path, searches in Folders.ThisAppImages; see also IconFlags.
string - image name (key) in the ImageList (System.Windows.Forms.ToolStripItem.ImageKey).
int - image index in the ImageList (System.Windows.Forms.ToolStripItem.ImageIndex).
Icon, Image, Folders.FolderPath.
null (default) - no icon. If ExtractIconPathFromCode == true, extracts icon path from onClick code like Shell.TryRun(@"c:\path\file.exe") or Shell.TryRun(Folders.System + "file.exe").
"" - no icon.

Examples
Copy
var m = new AuMenu();
m["One"] = o => Print(o);
m["Two", @"icon file path"] = o => { Print(o); AuDialog.Show(o.ToString()); };
m.LastItem.ToolTipText = "tooltip";
m["Three"] = o => { Print(o.MenuItem.Checked); };
m.LastMenuItem.Checked = true;
m.ExtractIconPathFromCode = true;
m["notepad"] = o => Shell.TryRun(Folders.System + "notepad.exe"));
m.Show();
