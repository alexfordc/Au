Remarks
This function usually is used to run ActiveNew triggers for a window created before Triggers.Run. Here "run triggers" means "compare window properties etc with those specified in triggers and run actions of triggers that match". Normally such triggers don't run because the window is considered old. This function runs triggers as it was a new window. Triggers like ActiveNew and ActiveOnce will run once, as usually.
This function must be called while the main triggers thread is in Triggers.Run, for example from another trigger action. It is asynchronous (does not wait).
If you call this function from another trigger action (hotkey etc), make sure the window trigger action runs in another thread or can be queed. Else both actions cannot run simultaneously. See example.
Examples
Copy
Triggers.Options.RunActionInNewThread(true);
Triggers.Window.ActiveNew["* Notepad"] = o => o.Window.Resize(500, 200);
Triggers.Hotkey["Ctrl+T"] = o => Triggers.Window.SimulateActiveNew(Wnd.Active);
Triggers.Hotkey["Ctrl+Alt+T"] = o => Triggers.Stop();
Triggers.Run();
