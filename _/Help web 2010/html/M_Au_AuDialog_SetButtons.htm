Parameters

buttons
Type: SystemString
Common and/or custom buttons, like with AuDialog.Show.
These ids should be negative if you use customButtons too, because ids of customButtons are 1, 2, ... .
asCommandLinks (Optional)
Type: SystemBoolean
Custom buttons style. If false - row of classic buttons. If true - column of command-link buttons that can have multiline text.
customButtons (Optional)
Type: Au.TypesMultiString
Additional custom buttons. All will be custom, even if named "OK" etc.
List of labels without ids. Can be string like "One|Two|..." or string[] or List<string>.
Button ids will be 1, 2, ... .
AuDialog.DefaultButton will be 1. You can change it later.
