File_MiscCalculateDirectorySize Method

Au Class Library
Calls File_.EnumDirectory and returns sum of all file sizes.
With default flags, it includes sizes of all descendant files, in this directory and all subdirectories except in inaccessible [sub]directories.

Remarks
This function is slow if the directory is large.
Don't use this function for files (throws exception) and drives (instead use System.IO.DriveInfo, it's fast and includes sizes of Recycle Bin and other protected hidden system directories).
