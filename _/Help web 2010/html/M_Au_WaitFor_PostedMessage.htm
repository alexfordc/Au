Parameters

secondsTimeout
Type: SystemDouble
The maximal time to wait, seconds. If 0, waits infinitely. If >0, after that time interval throws System.TimeoutException. If <0, after that time interval returns false.
callback
Type: Au.TypesWaitMsgCallback
Callback function that returns true to stop waiting. More info in Remarks.

Return Value

Type: Boolean
Returns true. On timeout returns false if secondsTimeout is negative; else exception.
Exceptions
ExceptionCondition
TimeoutExceptionsecondsTimeout time has expired (if > 0).
Remarks
While waiting, dispatches Windows messages etc, like Time.SleepDoEvents. Before dispatching a posted message, calls the callback function. Stops waiting when it returns true. Does not dispatch the message if the function sets the message field = 0.
Does not use Opt.WaitFor.
Examples
Copy
Timer_.After(2000, t => { Print("timer"); });
WaitFor.PostedMessage(5, (ref Native.MSG m) => { Print(m); return m.message == 0x113; }); //WM_TIMER
Print("finished");
