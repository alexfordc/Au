Remarks
Each thread has its own Opt.Key instance. It inherits options from Opt.Static.Key.
Also can be used when creating Keyb instances. See the second example.
Examples
Copy
Opt.Key.KeySpeed = 20;
Key("Tab Ctrl+V");
Use a Keyb instance.
Copy
var k = new Keyb(Opt.Key); //create new Keyb instance and copy options from Opt.Key to it
k.Options.KeySpeed = 100; //changes option of k but not of Opt.Key
k.Add("Tab Ctrl+V").Send(); //uses options of k
