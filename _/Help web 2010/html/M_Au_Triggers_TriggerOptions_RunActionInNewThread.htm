Don't run if this action is already running. If false, multiple action instances can run paralelly in multiple threads.
Remarks
Use if need to run actions simultaneously with other actions or other instances of self, especially if the action is long-running (maybe 5 s and more).
The thread has System.Threading.ApartmentState.STA.
The RunActionInX functions are mutually exclusive: only the last called function is active.
