Runs a console program, waits until its process ends, and gets its output text.
Returns the process exit code. Usually a non-0 value means error.

Parameters

output
Type: SystemString
Receives the output text.
Console programs have two output text streams - standard output and standard error. This function gets both, and the error text is always after the output text.
file
Type: SystemString
Path or name of an .exe or .bat file. Can be:
Full path. Examples: @"C:\folder\x.exe", Folders.System + "x.exe", @"%Folders.System%\x.exe".
Filename, like "x.exe". The function calls File_.SearchPath.
Path relative to Folders.ThisApp. Examples: "x.exe", @"subfolder\x.exe", @".\subfolder\x.exe", @"..\another folder\x.exe".
Supports environment variables, like @"%TMP%\x.bat". See Path_.ExpandEnvVar.
args (Optional)
Type: SystemString
Command line arguments.
This function expands environment variables if starts with "%" or "\"%".
directory (Optional)
Type: SystemString
Working directory. Default - System.IO.Directory.GetCurrentDirectory of this process.
textEncoding (Optional)
Type: System.TextEncoding
Text encoding used to convert console's ANSI text to C# Unicode text.
