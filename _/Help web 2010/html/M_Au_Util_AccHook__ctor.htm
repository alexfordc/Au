The lowest event constant value in the range of events. Can be AccEVENT.MIN to indicate the lowest possible event value. Events reference: SetWinEventHook.
eventMax
Type: Au.TypesAccEVENT
The highest event constant value in the range of events. Can be AccEVENT.MAX to indicate the highest possible event value. If 0, uses eventMin.
hookProc
Type: SystemActionHookDataAccHookData
The hook procedure (function that handles hook events).
idProcess (Optional)
Type: SystemInt32
The id of the process from which the hook function receives events. If 0 - all processes on the current desktop.
idThread (Optional)
Type: SystemInt32
The native id of the thread from which the hook function receives events. If 0 - all threads.
