Submenu(String, Object, ActionMTClickArgs)
Adds new item (System.Windows.Forms.ToolStripMenuItem) that will open a submenu.
Then the add-item functions will add items to the submenu.
Can be used in 2 ways:
1. using(m.Submenu(...)) { add items; }. See example.
2. m.Submenu(...); add items; m.EndSubmenu();. See AuMenu.EndSubmenu.
Submenu(String, ActionAuMenu, Object, ActionMTClickArgs)
Adds new item (System.Windows.Forms.ToolStripMenuItem) that will open a submenu.
When showing the submenu first time, your callback function will be called and can add submenu items.
