IntegrityLevel
Gets the UAC integrity level (IL) of the process.
IsAdmin
Returns true if this process is running as administrator, ie if the user belongs to the local Administrators group and the process is not limited by UAC.
This function for example can be used to check whether you can write to protected locations in the file system and registry.
IsUacDisabled
Returns true if UAC is disabled (turned off) completely (not just disabled UAC consent screeen/dialog).
IsUIAccess
Returns true if the process has uiAccess property.
A uiAccess process can access/automate all windows of processes running in the same user session.
