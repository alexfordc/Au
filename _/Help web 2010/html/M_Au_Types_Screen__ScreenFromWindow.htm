Screen_ScreenFromWindow Method

Au Class Library
Gets System.Windows.Forms.Screen object of the screen that contains the specified window (the biggest part of it) or is nearest to it.
If w handle is 0 or invalid, gets the primary screen (System.Windows.Forms.Screen.FromHandle would return an invalid object if the window handle is invalid).
