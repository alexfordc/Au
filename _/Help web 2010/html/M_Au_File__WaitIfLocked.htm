File_WaitIfLocked Method (Action, Int32)

Au Class Library
This function can be used to safely open a file that may be temporarily locked (used by another process or thread). Waits while the file is locked.

Remarks
This function calls the lambda and handles IOException. If the exception indicates that the file is locked, waits and retries in loop.
