Parameters

navig
Type: SystemString
String consisting of one or more navigation direction strings separated by space, like "parent next child4 first".
  • "next" - next sibling AO in the same parent AO.
  • "previous" - previous sibling AO in the same parent AO.
  • "first" - first child AO.
  • "last" - last child AO.
  • "parent" - parent (container) AO.
  • "child" - child AO by 1-based index. Example: "child3" (3-th child). Negative index means from end, for example -1 is the last child.
  • "#N" - N is a numeric value to pass to IAccessible.accNavigate as navDir. Can be any standard or custom value supported by the AO.
Can be only 2 letters, like "pr" for "previous".
A string like "next3" or "next,3" is the same as "next next next". Except for "child".

For "next", "previous", "firstchild", "lastchild", "#N" is used IAccessible.accNavigate. Not all AOs support it. Some AOs skip invisible siblings. Instead you can use "parent childN" or "childN".
For "parent" is used IAccessible.get_accParent. Few AOs don't support. Some AOs return a different parent than in the AO tree.
For "child" is used API AccessibleChildren.
secondsToWait (Optional)
Type: SystemDouble
Wait for the wanted AO max this number of seconds. If negative, waits forever.
