Remarks
How this function is different from System.Drawing.Image.FromHbitmap:
1. Image.FromHbitmap usually creates bottom-up bitmap, which is incompatible with WinImage.Find. This function creates normal top-down bitmap, like new Bitmap(...), Bitmap.FromFile(...) etc do.
2. This function always creates bitmap of PixelFormat Format32bppRgb.
