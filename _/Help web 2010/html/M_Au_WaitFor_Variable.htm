Parameters

secondsTimeout
Type: SystemDouble
The maximal time to wait, seconds. If 0, waits infinitely. If >0, after that time interval throws System.TimeoutException. If <0, after that time interval returns false.
variable
Type: SystemBoolean
Stop waiting when this variable is set to true.
options (Optional)
Type: Au.TypesOptWaitFor
Options. If null, uses Opt.WaitFor, else combines with it.

Return Value

Type: Boolean
Returns true. On timeout returns false if secondsTimeout is negative; else exception.
Exceptions
ExceptionCondition
TimeoutExceptionsecondsTimeout time has expired (if > 0).
Remarks
This function is useful when the variable can be changed by any thread. To wait for a variable changed while processing messages etc in this thread, it's better to use WaitFor.MessagesAndCondition.
Examples
Copy
bool stop = false;
Task.Run(() => { 2.s(); Print("task"); stop = true; });
WaitFor.Variable(5, stop);
Print(stop);
