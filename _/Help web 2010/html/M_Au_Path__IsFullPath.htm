Remarks
Returns true if path matches one of these wildcard patterns:
  • @"?:\*" - local path, like @"C:\a\b.txt". Here ? is A-Z, a-z.
  • @"?:" - drive name, like @"C:". Here ? is A-Z, a-z.
  • @"\\*" - network path, like @"\\server\share\...". Or has prefix @"\\?\".
Supports '/' characters too.
Supports only file-system paths. Returns false if path is URL (Path_.IsUrl) or starts with "::".
If path starts with "%environmentVariable%", shows warning and returns false. You should at first expand environment variables with Path_.ExpandEnvVar or instead use Path_.IsFullPathExpandEnvVar.
