The AccFinder type exposes the following members.

Properties
  NameDescription
NavigFailed
true if used parameter navig and the intermediate object was found but the navigation did not find the final object.
Result
The found accessible object.
null if not found. null if used Acc.Finder.ResultGetProperty.
ResultGetProperty
Set this when you need only some property of the accessible object (name, etc) and not the object itself.
The value is a character, the same as with Acc.GetProperties, for example 'n' for Name. Use '-' if you don't need any property.
ResultProperty
The requested propery of the found accessible object, depending on Acc.Finder.ResultGetProperty.
null if: 1. Object not found. 2. ResultGetProperty not used or is '-'. 3. Failed to get the property.
