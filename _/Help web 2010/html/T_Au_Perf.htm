Incremental
If true, times of each new First/Next/Next... measurement are added to previous measurement times.
Finally you can call Perf.Write or Perf.ToString to get the sums.
Usually used to measure code in loops. See example.
TimeTotal
Gets the number of microseconds between Perf.First and the last Perf.Next.
