Finds a child control by its id and returns its handle as Wnd.
Returns default(Wnd) if not found. See also: Wnd.Is0, OrThrow(Wnd).

This function supports flags DirectChild and HiddenToo. If both are set, it is much faster because uses API GetDlgItem. Else uses API EnumChildWindows, like Wnd.Child.

Type: Wnd
Exceptions

WndExceptionThis variable is invalid (window not found, closed, etc).
Remarks
To create code for this function, use dialog "Find window or control". It is form Au.Tools.Form_Wnd in Au.Tools.dll.

Not all controls have a useful id. If control id is not unique or is different in each window instance, this function is not useful.
