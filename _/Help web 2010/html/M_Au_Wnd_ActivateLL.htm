WndActivateLL Method

Au Class Library
Low-level version of Wnd.Activate.
Just calls Wnd.Misc.EnableActivate, API SetForegroundWindow and makes sure that it actually worked, but does not check whether it activated exactly this window.
No exceptions, does not unhide, does not restore minimized, does not check is it a top-level window or control, etc.
Returns false if fails.
