Remarks
Uses IAccessible.accSelect.
Not all objects support it. Most objects support not all flags. It depends on object states FOCUSABLE, SELECTABLE, MULTISELECTABLE, EXTSELECTABLE, DISABLED.
Many object have bugs, especially with flag TAKEFOCUS. More bugs when the object found with flag AFFlags.NotInProc.
