useLastXY (Optional)
Type: SystemBoolean
Use Mouse.LastXY. It is the mouse cursor position set by the most recent 'mouse move' or 'mouse click' function called in this thread. Use this option for reliability.
Example: Mouse.Move(100, 100); Mouse.ClickEx(..., true);. The click is always at 100 100, even if somebody changes cursor position between Mouse.Move sets it and Mouse.ClickEx uses it. In such case this option atomically moves the cursor to LastXY. This movement is instant and does not use Opt.
If false (default), clicks at the current cursor position (does not move it).

Return Value

Type: MRelease
The return value can be used to auto-release pressed button. See example with Mouse.LeftDown.
