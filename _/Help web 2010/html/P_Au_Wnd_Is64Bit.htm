WndIs64Bit Property

Au Class Library
Returns true if the window is of a 64-bit process, false if of a 32-bit process.
Also returns false if fails. Supports WinError.Code.
If Ver.Is64BitOS is true, calls API GetWindowThreadProcessId, OpenProcess and IsWow64Process.
 Note
If you know that the window belongs to current process, instead use System.Environment.Is64BitProcess or IntPtr.Size==8. This function is much slower.
