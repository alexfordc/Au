Remarks
Similar to System.IO.Path.GetFileName and System.IO.Path.GetFileNameWithoutExtension. Some diferences: does not throw exceptions; if ends with '\\' or '/', gets part before it, eg "B" from @"C:\A\B\".
Supports separators '\\' and '/'.
Also supports URL and shell parsing names like @"::{CLSID-1}\0\::{CLSID-2}".
Example paths and results:
Copy
@"C:\A\B\file.txt" -> "file.txt".
"file.txt" -> "file.txt".
"file" -> "file".
@"C:\A\B" -> "B".
@"C:\A\B\" -> "B".
@"C:\A\/B\/" -> "B".
@"C:\" -> "".
@"C:" -> "".
@"\\network\share" -> "share".
@"C:\aa\file.txt:alt.stream" -> "file.txt:alt.stream".
"http://a.b.c" -> "a.b.c".
"::{A}\::{B}" -> "::{B}".
"" -> "".
null -> null.
Example paths and results when withoutExtension true:
Copy
@"C:\A\B\file.txt" -> "file".
"file.txt" -> "file".
"file" -> "file".
@"C:\A\B" -> "B".
@"C:\A\B\" -> "B".
@"C:\A\B.B\" -> "B.B".
@"C:\aa\file.txt:alt.stream" -> "file.txt:alt".
"http://a.b.c" -> "a.b".
