EnabledAlways
If true, triggers added afterwards don't depend on Triggers.ActionTriggers.Disabled and Triggers.ActionTriggers.DisabledEverywhere.
This property sets the Triggers.ActionTrigger.EnabledAlways property of triggers added afterwards.
