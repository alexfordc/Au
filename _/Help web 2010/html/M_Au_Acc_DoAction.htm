Remarks
Fails if the object does not have a default action. Then you can use MouseClick(Acc, Coord, Coord, MButton), or try Acc.VirtualClick, Acc.Select, Acc.Focus and keyboard functions.
The action can take long time, for example show a dialog. This function normally does not wait. It allows the caller to automate the dialog. If it waits, try Acc.DoJavaAction or one of the above functions (MouseClick etc).
