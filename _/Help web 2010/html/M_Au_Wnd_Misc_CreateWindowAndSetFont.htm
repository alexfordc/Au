WndMiscCreateWindowAndSetFont Method

Au Class Library
Creates native/unmanaged window like Wnd.Misc.CreateWindow and sets font.
If customFontHandle not specified, sets the system UI font, usually it is Segoe UI, 9.
Later call Wnd.Misc.DestroyWindow or Wnd.Close.
