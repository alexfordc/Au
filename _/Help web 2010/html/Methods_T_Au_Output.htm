Output Methods

Au Class Library

The Output type exposes the following members.

Methods
  NameDescription
Clear
Clears the output window or console text (if Output.IsWritingToConsole) or log file (if Output.LogFile not null).
Write
Writes string + "\r\n" to the output window or console (if Output.IsWritingToConsole) or log file (if Output.LogFile not null).
WriteDirectly
Writes string value + "\r\n" to the output window or console (if Output.IsWritingToConsole) or log file (if Output.LogFile not null).
Unlike Output.Write, Print etc, the string is not passed to Output.Writer.
