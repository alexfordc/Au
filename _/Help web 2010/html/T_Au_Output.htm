IgnoreConsole
If true, Write and related functions in console process don't not use the console window. Then everything is like in non-console process.
IsConsoleProcess
Returns true if this is a console process.
IsWritingToConsole
Returns true if is writing to console, false if to the output window or log file. Assuming that Output.Writer is not changed.
LogFile
Sets log file path.
When set (not null), text passed to Output.Write, Print and similar functions will be written to the file. Assuming that Output.Writer is not changed.
If value is null - restores default behavior.
LogFileTimestamp
Let Write etc also add current time when using log file (see Output.LogFile).
The time is local, not UTC.
RedirectConsoleOutput
Let Console.WriteX methods in non-console process write to the same destination as Output.Write etc.
RedirectDebugOutput
Let Debug.WriteX and Trace.WriteX methods write to the same destination as Output.Write etc.
Writer
Gets or sets object that actually writes text when is called Output.Write, Print and similar functions.

Remarks
When Output.Write, Print, etc is called, where the text goes:
  • If redirected, to wherever it is redirected. See Output.Writer.
  • Else if using log file (Output.LogFile not null), writes to the file.
  • Else if using console (Output.IsWritingToConsole returns true), writes to console.
  • Else if using local Util.OutputServer (in this appdomain), writes to it.
  • Else if exists global Util.OutputServer (in any process/appdomain), writes to it.
  • Else nowhere.
