Remarks
This class adds these features:
1. Static constructor subscribes to System.AppDomain.UnhandledException event. On unhandled exception prints exception info. Without this class not all unhandled exceptions would be printed.
2. Provides function OnUnhandledException. The script can override it.

More features may be added in the future.
