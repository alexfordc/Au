Parameters

p
Type: Au.TypesPOINT
Coordinates relative to the primary screen.
Tip: When need coordinates relative to another screen or/and the work area, use Coord.Normalize or tuple (x, y, workArea) etc. Example: Mouse.Move((x, y, true));.
Exceptions
ExceptionCondition
ArgumentOutOfRangeExceptionThe specified x y is not in screen. No exception if option Relaxed is true (then moves to a screen edge).
AuExceptionFailed to move the cursor to the specified x y. Some reasons: 1. Another thread blocks or modifies mouse input (API BlockInput, mouse hooks, frequent API SendInput etc); 2. The active window belongs to a process of higher UAC integrity level; 3. Some application called API ClipCursor. No exception if option Relaxed is true (then final cursor position is undefined).
Remarks
Uses Opt.Mouse: MoveSpeed, MoveSleepFinally, Relaxed.
Examples
Save-restore mouse position.
Copy
var p = Mouse.XY;
//...;
Mouse.Move(p);
Use coodinates in the first non-primary screen.
Copy
Mouse.Move(Coord.Normalize(10, 10, screen: 1));
