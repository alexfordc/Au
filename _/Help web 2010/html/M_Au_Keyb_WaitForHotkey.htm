Registers a temporary hotkey and waits for it.

Namespace:  Au
Assembly:  Au (in Au.dll) Version: 1.0.0.0
Syntax
C#
Copy
public static bool WaitForHotkey(
	double secondsTimeout,
	KHotkey hotkey,
	bool waitModReleased = false
)

Parameters

secondsTimeout
Type: SystemDouble
The maximal time to wait, seconds. If 0, waits infinitely. If >0, after that time interval throws System.TimeoutException. If <0, after that time interval returns false.
hotkey
Type: Au.TypesKHotkey
Hotkey. Can be: string like "Ctrl+Shift+Alt+Win+K", tuple (KMod, KKey), enum KKey, enum Keys, struct KHotkey.
waitModReleased (Optional)
Type: SystemBoolean
Also wait until hotkey modifier keys released.

Return Value

Type: Boolean
Returns true. On timeout returns false if secondsTimeout is negative; else exception.
Exceptions
ExceptionCondition
ArgumentExceptionError in hotkey string.
AuExceptionFailed to register hotkey.
TimeoutExceptionsecondsTimeout time has expired (if > 0).
Remarks
Uses Util.RegisterHotkey; it uses API RegisterHotKey.
Fails if the hotkey is currently registered by this or another application or used by Windows. Also if F12.
Examples
Copy
Keyb.WaitForHotkey(0, "F11");
Keyb.WaitForHotkey(0, KKey.F11);
Keyb.WaitForHotkey(0, "Shift+A", true);
Keyb.WaitForHotkey(0, (KMod.Ctrl | KMod.Shift, KKey.P)); //Ctrl+Shift+P
Keyb.WaitForHotkey(0, Keys.Control | Keys.Alt | Keys.H); //Ctrl+Alt+H
Keyb.WaitForHotkey(5, "Ctrl+Win+K"); //exception after 5 s
if(!Keyb.WaitForHotkey(-5, "Left")) Print("timeout"); //returns false after 5 s
