Parameters

renderLater (Optional)
Type: SystemBoolean
Call API SetClipboardData(format, default). When/if some app will try to get clipboard data, the first time your clipboard owner window will receive WM_RENDERFORMAT message and should call SetOpenClipboard(false);.
format (Optional)
Type: SystemInt32
Copy data only of this format. If 0 (default), of all formats.
Exceptions
ExceptionCondition
OutOfMemoryExceptionFailed to allocate memory for clipboard data.
AuExceptionFailed to set clipboard data.
Remarks
This function is similar to Clipb.Data.SetClipboard. It calls API SetClipboardData and does not call OpenClipboard, EmptyClipboard, CloseClipboard. The clipboard must be open and owned by a window of this thread.
