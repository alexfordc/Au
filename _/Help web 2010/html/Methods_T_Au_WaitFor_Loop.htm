Methods
  NameDescription
IsTimeout
If the secondsTimeout time is not expired, returns false.
Else if secondsTimeout is negative, returns true.
Else throws System.TimeoutException.
Sleep
Calls WaitFor.Loop.IsTimeout. If it returns true, returns false.
Else sleeps for WaitFor.Loop.Period milliseconds, increments Period if it is less than WaitFor.Loop.MaxPeriod, and returns true.
