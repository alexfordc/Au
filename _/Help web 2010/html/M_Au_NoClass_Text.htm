Sends text to the active window, using virtual keystrokes or the clipboard. Then also can send non-text keystrokes.
Calls Keyb.Text.

Parameters

text
Type: SystemString
Text to send.
keysEtc
Type: SystemObject
Optional more parameters. The same as with Keyb.Key. Can be used for example to press non-text keys, wait, send more text.
Exceptions
ExceptionCondition
ArgumentExceptionAn argument in keysEtc is of an unsupported type or has an invalid value, for example an unknown key name.
Remarks
This function is identical to Keyb.Key, except: the first parameter is literal text (not keys). This example shows the difference: Key("keys", "text", "keys", "text"); Text("text", "keys", "text", "keys");.
To send text can be used keys or clipboard, depending on Opt.Key and text.
More info in Keyb.Key topic.
Examples
Copy

Keyb.Text("Send this text, press key", "Enter", "and wait", 500, "milliseconds. Enter");
Text("Can be used without the \"Keyb.\" prefix.\n");
