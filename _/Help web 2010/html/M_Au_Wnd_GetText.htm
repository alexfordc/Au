WndGetText Method

Au Class Library
Gets window/control name or control text.
Returns "" if it is empty.
Returns null if fails, eg if the window is closed. Supports WinError.Code.
This is a low-level function. You can instead use Wnd.Name and Wnd.ControlText.

Parameters

getText (Optional)
Type: SystemNullableBoolean
false - use API function InternalGetWindowText. It is fast and usually does not get variable text. This is used by Wnd.Name.
true - use API message WM_GETTEXT. It is slow and prefers variable text. This is used by Wnd.ControlText. Fails if the window is hung.
null - try InternalGetWindowText. If it gets "", and this is a control, then try WM_GETTEXT.
removeUnderlineAmpersand (Optional)
Type: SystemBoolean
Remove the invisible '&' characters that are used to underline keyboard shortcuts with the Alt key.
Removes only if this is a control (has style WS.CHILD).
Calls Util.StringMisc.RemoveUnderlineAmpersand.
