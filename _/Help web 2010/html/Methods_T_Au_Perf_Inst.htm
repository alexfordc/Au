ToArray
Return array of time values collected by calling Perf.Inst.First and Perf.Inst.Next.
Each element is the number of microseconds of each code execution between calling First and each Next.
ToString
Formats a string from time values collected by calling Perf.Inst.First and Perf.Inst.Next.
The string contains the number of microseconds of each code execution between calling First and each Next.
(Overrides ValueTypeToString.)
Write
Formats a string from time values collected by calling Perf.Inst.First and Perf.Inst.Next, and shows it in the output.
The string contains the number of microseconds of each code execution between calling First and each Next.
