AuMenuSubmenu Method (String, ActionAuMenu, Object, ActionMTClickArgs)

Au Class Library
Adds new item (System.Windows.Forms.ToolStripMenuItem) that will open a submenu.
When showing the submenu first time, your callback function will be called and can add submenu items.

Parameters

text
Type: SystemString
Text.
onOpening
Type: SystemActionAuMenu
Callback function that should add submenu items.
icon (Optional)
Type: SystemObject
Can be:
string - path of .ico or any other file or folder or non-file object. See Icon_.GetFileIcon. If not full path, searches in Folders.ThisAppImages; see also IconFlags.
string - image name (key) in the ImageList (System.Windows.Forms.ToolStripItem.ImageKey).
int - image index in the ImageList (System.Windows.Forms.ToolStripItem.ImageIndex).
Icon, Image, Folders.FolderPath.
null (default) - no icon. If ExtractIconPathFromCode == true, extracts icon path from onClick code like Shell.TryRun(@"c:\path\file.exe") or Shell.TryRun(Folders.System + "file.exe").
"" - no icon.
onClick (Optional)
Type: SystemActionMTClickArgs
Callback function. Called when the item clicked. Rarely used.

Examples
Copy
var m = new AuMenu();
m["One"] = o => Print(o);
m["Two"] = o => Print(o);
m.Submenu("Submenu 1", m1 =>
{
    Print("adding items of " + m.CurrentAddMenu.OwnerItem);
    m["Three"] = o => Print(o);
    m["Four"] = o => Print(o);
    m.Submenu("Submenu 2", m2 =>
    {
        Print("adding items of " + m.CurrentAddMenu.OwnerItem);
        m["Five"] = o => Print(o);
        m["Six"] = o => Print(o);
    });
    m["Seven"] = o => Print(o);
});
m["Eight"] = o => Print(o);
m.Show();
