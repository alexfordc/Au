public enum OsdShowMode
Members
  Member nameValueDescription
Auto0Depends on Thread_.HasMessageLoop. If it is true, uses ThisThread, else StrongThread. Does not wait.
ThisThread1 Show the OSD window in this thread and don't wait.
Should not be used if this thread does not process messages and therefore cannot have windows.
WeakThread2Show the OSD window in new thread and don't wait. Set System.Threading.Thread.IsBackground=true, so that the OSD is closed when other threads of this app end.
StrongThread3Show the OSD window in new thread and don't wait. Set System.Threading.Thread.IsBackground=false, so that the OSD is not closed when other threads of this app end.
Wait4 Show the OSD window in this thread and wait until it disappears.
Waits Osd.SecondsTimeout seconds. While waiting, dispatches messages etc; see Time.SleepDoEvents.
Remarks
If this thread has windows, any value can be used, but usually Auto (default) or ThisThread is the best.
