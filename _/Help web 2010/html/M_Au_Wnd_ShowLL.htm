Remarks
This library has two similar functions - Wnd.Show and ShowLL. Show is better to use in automation scripts, with windows of any process/thread. ShowLL usually is better to use in programming, with windows of current thread.
ShowLL is more low-level. Does not throw exception when fails, and does not add a delay; Show adds a small delay when the window is of other thread.

Calls API ShowWindow with SW_SHOWNA or SW_HIDE.
Supports WinError.Code.
