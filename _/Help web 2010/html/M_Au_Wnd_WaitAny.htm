WndWaitAny Method

Au Class Library
Waits until any of specified windows exists, is visible (optionally) and active (optionally).
Returns window handle. On timeout returns default(Wnd) if secondsTimeout is negative; else exception.

Parameters

secondsTimeout
Type: SystemDouble
The maximal time to wait, seconds. If 0, waits infinitely. If >0, after that time interval throws System.TimeoutException. If <0, after that time interval returns default(Wnd).
active
Type: SystemBoolean
The window must be the active window (Wnd.Active), and not minimized.
windows
Type: AuWndFinder
One or more variables containing window properties. Can be strings, see Wnd.Finder.op_Implicit.

Return Value

Type: Wnd
