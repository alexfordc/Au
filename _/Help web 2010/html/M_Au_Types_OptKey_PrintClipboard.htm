public static void PrintClipboard()
Remarks
Shows this info in the output, for each clipboard format: format name, time spent to get data (microseconds), data size (bytes), and whether this format would be restored (depends on OptKey.RestoreClipboardExceptFormats).
 Note
Copy something to the clipboard each time before calling this function. Don't use Clipb.CopyText and don't call this function in loop. Else it shows small times.
The time depends on app, etc. More info: OptKey.RestoreClipboardExceptFormats.
