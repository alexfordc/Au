Can optionally specify one or more additional events.
This starts to work when the primary trigger is activated, and works only for that window.
For example, to be notified when the window is closed or renamed, specify later: TWEvents.Destroyed | TWEvents.Name.
When a "later" event occurs, the trigger action is executed. The Triggers.WindowTriggerArgs.Later property then is that event; it is 0 when it is the primary trigger.
The "later" trigers are not disabled when primary triggers are disabled.
