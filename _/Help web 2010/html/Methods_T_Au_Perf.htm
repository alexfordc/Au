Perf Methods

Au Class Library

The Perf type exposes the following members.

Methods
  NameDescription
Cpu
Executes some code in loop for the specified amount of time. It should make CPU to run at full speed.
First
Stores current time in the first element of an internal array.
Next
Stores current time in next element of an internal array.
NW
Calls Perf.Next and Perf.Write.
StartNew
Creates and returns new Perf.Inst variable and calls its Perf.Inst.First.
ToArray
Return array of time values collected by calling Perf.First and Perf.Next.
Each element is the number of microseconds of each code execution between calling First and each Next.
ToString
Formats a string from time values collected by calling Perf.First and Perf.Next.
The string contains the number of microseconds of each code execution between calling First and each Next.
Write
Formats a string from time values collected by calling Perf.First and Perf.Next, and shows it in the output.
The string contains the number of microseconds of each code execution between calling First and each Next.
