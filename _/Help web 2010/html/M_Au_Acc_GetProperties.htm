Remarks
The returned variable contains values of properties specified in props. When a property is empty or failed to get, the member variable is "", empty dictionary or default value of that type; never null.

Normally this function is faster than calling multiple property functions, because it makes single remote procedure call. But not if this accessible object was found with flag AFFlags.NotInProc etc.

Returns false if fails, for example when the object's window is closed. Supports WinError.Code.
