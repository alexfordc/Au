Parameters

recorded
Type: System.Collections.GenericIEnumerableUInt32
List of x y distances from previous.
The first distance is from the mouse position before the first movement; at run time it will be distance from Mouse.LastXY.
To create uint value from distance dx dy use this code: Math_.MakeUint(dx, dy).
withSleepTimes
Type: SystemBoolean
recorded also contains sleep times (milliseconds) alternating with distances.
It must start with a sleep time. Example: {time1, dist1, time2, dist2}. Another example: {time1, dist1, time2, dist2, time3}. This is invalid: {dist1, time1, dist2, time2}.
