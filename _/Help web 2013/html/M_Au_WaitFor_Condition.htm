Waits for an user-defined condition.

Namespace:  Au

public static bool Condition(
	double secondsTimeout,
	Func<bool> condition,
	OptWaitFor options = null
)

Parameters

secondsTimeout
Type: SystemDouble
The maximal time to wait, seconds. If 0, waits infinitely. If >0, after that time interval throws System.TimeoutException. If <0, after that time interval returns false.
condition
Type: SystemFuncBoolean
Callback function (eg lambda). It is called repeatedly, until returns true. The calling period depends on options.
options (Optional)
Type: Au.TypesOptWaitFor
Options. If null, uses Opt.WaitFor, else combines with it.

Return Value

Type: Boolean
Returns true. On timeout returns false if secondsTimeout is negative; else exception.

TimeoutExceptionsecondsTimeout time has expired (if > 0).
