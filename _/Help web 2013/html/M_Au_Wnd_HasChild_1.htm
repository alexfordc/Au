WndHasChild Method (String, String, WCFlags, FuncWnd, Boolean, Int32)

Returns true if this window contains the specified control.
Calls Wnd.Child.
 Note
Using this function many times with same parameters is inefficient. Instead create new Wnd.ChildFinder and call Wnd.ChildFinder.Find or Wnd.HasChild. See example.
