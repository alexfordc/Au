Remarks
Used mostly for function parameters. The caller can specify screen index (int), window (Wnd etc), System.Windows.Forms.Screen object, etc. There are implicit conversion operators from these types. The Screen_ variable holds the specified value. When the function needs screen properties, it calls Screen_.GetScreen to get System.Windows.Forms.Screen object corresponding that value at that time.
