Remarks
Most 'wait for' functions of this library use WaitFor.Loop, which repeatedly checks the wait condition and sleeps (waits) several ms. This property sets the initial sleep time, which then is incremented by Period/10 ms (default 1 ms) in each loop until reaches Period*50 (default 500 ms).
This property makes the response time shorter or longer. If <10, makes it shorter (faster response), but increases CPU usage; if >10, makes it longer (slower response).
