OptKeyKeySpeedClipboard Property

How long to wait (milliseconds) between sending Ctrl+V and Ctrl+C keys of clipboard functions (paste, copy).
Default: 5. Valid values: 0 - 1000 (1 second). Valid values for Opt.Static.Key: 0 - 50.
