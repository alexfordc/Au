Remarks
Can be used instead of Wnd.Find or Wnd.FindAll.
These codes are equivalent:
Copy
Wnd w = Wnd.Find(a, b, c, d, e); if(!w.Is0) Print(w);
Copy
var p = new Wnd.Finder(a, b, c, d, e); if(p.Find()) Print(p.Result);
Also can find in a list of windows.
