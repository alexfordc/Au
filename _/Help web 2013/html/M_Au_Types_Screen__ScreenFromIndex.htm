Parameters

screenIndex
Type: SystemInt32
Values greater than 0 are used for non-primary screens: 1 - first non-primary, 2 second...
Also can be Screen_.Primary (0), Screen_.OfMouse (-1), Screen_.OfActiveWindow (-2).
If invalid, prints warning and gets the primary screen.

Return Value

Type: Screen
Remarks
Uses System.Windows.Forms.Screen.AllScreens to get screen indices. They may not match the indices that you can see in Windows Settings.
