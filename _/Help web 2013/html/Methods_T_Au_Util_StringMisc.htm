Methods
  NameDescription
CommandLineFromArray
Converts array of command line arguments to string that can be passed to a "start process" function, for example Shell.Run, System.Diagnostics.Process.Start.
Returns null if a is null or has 0 elements.
CommandLineToArray
Parses command line arguments.
Calls API CommandLineToArgvW.
Returns empty array if s is null or "".
ParseIntAndString_
If string contains a number at startIndex, gets that number as int, also gets the string part that follows it, and returns true.
For example, for string "25text" or "25 text" gets num = 25, tail = "text".
Everything else is the same as with String_.ToInt_.
RemoveUnderlineAmpersand
Removes '&' characters from string.
Replaces "&&" with "&".
Returns new string if s has '&' characters, else returns s.
StringToIntArray
Creates int[] from string containing space-separated numbers, like "4 100 -8 0x10".
