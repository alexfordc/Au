AuMenuUsingSubmenu
Allows to use code: using(m.Submenu("Name")) { add items; } .
Convert_MD5Hash
Computes MD5 hash of some data.
Multiple datas can be hashed, producing single result.
Call Add one or more times. Finally use Convert_.MD5Hash.Hash to get result.
Convert_MD5HashResult
Result of Convert_.MD5Hash.Hash.
It is 16 bytes stored in 2 long fields r1 and r2.
If need, can be converted to byte[] with Convert_.MD5HashResult.ToArray or to hex string with Convert_.MD5HashResult.ToString.
File_MiscDisableRedirection
Temporarily disables file system redirection, to allow this 32-bit process access the 64-bit System32 directory.
File_MiscFileId
