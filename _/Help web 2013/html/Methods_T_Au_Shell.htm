Shell Methods

The Shell type exposes the following members.

Methods
  NameDescription
Run
Runs/opens a program, document, directory (folder), URL, new email, Control Panel item etc.
The returned SRResult variable contains some process info - process id etc.
RunConsole(String, String, String, Encoding)
Runs a console program (hidden), waits until its process ends, and prints its output text.
Calls Shell.RunConsole and Print(String).
RunConsole(String, String, String, String, Encoding)
Runs a console program, waits until its process ends, and gets its output text.
Returns the process exit code. Usually a non-0 value means error.
SelectFileInExplorer
Opens parent folder in Explorer and selects the file.
Returns null if fails, for example if the file does not exist.
TryRun
Calls Shell.Run and handles exceptions. All parameters are the same.
If Run throws exception, prints it as warning and returns null.
