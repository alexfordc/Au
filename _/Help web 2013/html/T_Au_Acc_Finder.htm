Properties

NavigFailed
true if used parameter navig and the intermediate object was found but the navigation did not find the final object.
Result
The found accessible object.
null if not found. null if used Acc.Finder.ResultGetProperty.
ResultGetProperty
Set this when you need only some property of the accessible object (name, etc) and not the object itself.
The value is a character, the same as with Acc.GetProperties, for example 'n' for Name. Use '-' if you don't need any property.
ResultProperty
The requested propery of the found accessible object, depending on Acc.Finder.ResultGetProperty.
null if: 1. Object not found. 2. ResultGetProperty not used or is '-'. 3. Failed to get the property.

Methods
  NameDescription
Find(Acc)
Finds accessible object (AO) in another AO, like Acc.Find.
Returns true if found. The Acc.Finder.Result property will be the found AO.
Find(Wnd)
Finds accessible object (AO) in window w, like Acc.Find.
Returns true if found. The Acc.Finder.Result property will be the found AO.
Find(Wnd, WndChildFinder)
Finds accessible object (AO) in the specified control of window w.
Returns true if found. The Acc.Finder.Result property will be the found AO.
Wait(Double, Acc)
Finds accessible object (AO) in another AO.
The same as Acc.Finder.Find, but waits until the AO is found or the given time expires.
Wait(Double, Wnd)
Finds accessible object (AO) in window w.
The same as Acc.Finder.Find, but waits until the AO is found or the given time expires.

Remarks
Can be used instead of Acc.Find.
Examples
Find window that contains certain AO, and get the AO too.
Copy
var f = new Acc.Finder("BUTTON", "Apply"); //AO properties
Wnd w = Wnd.Find(cn: "#32770", also: t => f.Find(t));
Print(w);
Print(f.Result);
