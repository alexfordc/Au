Parameters

onlyVisible (Optional)
Type: SystemBoolean
Need only visible windows.
Note: this function does not check whether windows are cloaked, as it is rather slow. Use Wnd.IsCloaked if need.
sortFirstVisible (Optional)
Type: SystemBoolean
Place hidden windows at the end of the array. If false, the order of array elements matches the Z order.
Not used when onlyVisible is true.

Return Value

Type: Wnd
Remarks
Calls API EnumWindows.
 Note
The array can be bigger than you expect, because there are many invisible windows, tooltips, etc. See also Wnd.GetWnd.MainWindows.
Does not get message-only windows. Use Wnd.Misc.FindMessageOnlyWindow if need.
On Windows 8 and later does not get Windows Store app Metro-style windows (on Windows 10 few such windows exist), unless this process has UAC integrity level uiAccess or High+uiAccess or its manifest contains disableWindowFiltering; to get such windows you can use Wnd.FindFast.
Tip: To get top-level and child windows in single array: var a = Wnd.GetWnd.Root.Get.Children();.
