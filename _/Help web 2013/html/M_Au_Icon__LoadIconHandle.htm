Parameters

.ico, .exe, .dll or other file that contains one or more icons. Also supports cursor files - .cur, .ani. Must be full path, without icon index. Supports environment variables (see Path_.ExpandEnvVar).
index (Optional)
Type: SystemInt32
Icon index or negative icon resource id in the .exe/.dll file.
size (Optional)
Type: SystemInt32
Icon width and height. Also can be enum IconSize, cast to int.

Returns icon handle, or default(IntPtr) if failed.
