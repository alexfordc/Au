AuMenu Properties

The AuMenu type exposes the following members.

Properties
  NameDescription
ActivateMenuWindow
Activate the menu window.
It enables selecting menu items with the keyboard (arrows, Tab, Enter, etc).
If false, only Esc works, it closes the menu.
If the menu is owned by a control or toolbar button, keyboard navigation works in any case, don't need this property to enable it.
CMS
Gets ContextMenuStrip that is used to show the main drop-down menu.
You can use all its properties, methods and events. You can assign it to a control or toolstrip's drop-down button etc.
CurrentAddMenu
Gets System.Windows.Forms.ToolStripDropDownMenu of the main menu or submenu where new items currently are added.
DefaultActivateMenuWindow
Default AuMenu.ActivateMenuWindow value for new AuMenu instances.
DefaultMouseClosingDistance
Default MouseClosingDistance value of AuMenu instances.
A AuMenu instance inherits this at the moment it is created.
DefaultMultiShow
Default AuMenu.MultiShow value for new AuMenu instances.
ExceptionHandling
Whether/how to handle unhandled exceptions in item code.
Default: MTExcept.Exception (don't handle exceptions if BaseMT.ItemThread is MTThread.Current (default), else show warning).
(Inherited from BaseMT.)
ExtractIconPathFromCode
When adding items without explicitly specified icon, extract icon from item code.
(Inherited from BaseMT.)

(Inherited from BaseMT.)

IsDisposed
Item
Adds new item as System.Windows.Forms.ToolStripMenuItem.
Sets its text, icon and Click event handler. Other properties can be specified later. See example.
Code m.Add("text", o => Print(o)); is the same as m["text"] = o => Print(o); .
ItemThread
In what thread to execute item callback functions.
Default: current thread.
(Inherited from BaseMT.)
LastItem
Gets the last added item as System.Windows.Forms.ToolStripItem, which is the base type of System.Windows.Forms.ToolStripMenuItem, System.Windows.Forms.ToolStripButton and other supported types.
(Inherited from BaseMT.)
LastMenuItem
Gets the last added item as System.Windows.Forms.ToolStripMenuItem.
Returns null if it is not a ToolStripMenuItem, for example a button or separator.
MainToolStrip
Infrastructure.
(Overrides BaseMTMainToolStrip.)
ModalAlways
If true, Show always waits until the menu is closed.
If false, does not wait if the thread has a .NET message loop (Thread_.HasMessageLoop).
MouseClosingDistance
Close the menu when the mouse cursor moves away from it to this distance, pixels.
At first the mouse must be or move at less than half of the distance.
Default is equal to AuMenu.DefaultMouseClosingDistance, default 200.
MultiShow
If false, disposes the menu when it is closed.
If true, does not dispose. Then you can call Show again and again.
Default is false, but is automatically set to true when showing the menu not with Show, eg when assigned to a control.
