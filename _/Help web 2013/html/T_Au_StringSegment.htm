Remarks
The main properties are: StringSegment.Buffer - the string containing the substring; StringSegment.Offset - the start index of the substring in the string; StringSegment.Length substring length. There is no string object for the substring itself; however StringSegment.Value and some other functions allocate and return a new string object.

One of ways to create StringSegment instances is to split a string with String_.Segments_.

Don't use the default constructor (parameterless). Then Buffer is null and the behavior of functions is undefined. Other constructors throw exception if the string is null.

Functions throw ArgumentOutOfRangeException, ArgumentNullException, NullReferenceException or IndexOutOfRangeException exception when an argument is invalid.
