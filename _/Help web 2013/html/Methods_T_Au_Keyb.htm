Keyb Methods

The Keyb type exposes the following members.

Methods
  NameDescription
Add
Adds keystrokes, text, sleep and other events to the internal collection. They will be sent/executed by Keyb.Send.
Returns self.
AddCallback
Adds a callback function.
Returns self.
AddKey(KKey, NullableBoolean)
Adds single key, specified as KKey, to the internal collection. It will be sent by Keyb.Send.
Returns self.
AddKey(KKey, Int32, Boolean, NullableBoolean)
Adds single key to the internal collection. Allows to specify scan code and whether it is an extended key. It will be sent by Keyb.Send.
Returns self.
AddKeys
Adds keystrokes to the internal collection. They will be sent by Keyb.Send.
Returns self.
AddRepeat
Adds the repeat-key operator. Then Keyb.Send will send the last added key the specified number of times.
Returns self.
AddSleep
Adds a short pause. Then Keyb.Send will sleep (wait).
Returns self.
AddText
Adds text. It will be sent by Keyb.Send.
Returns self.
GetMod
Gets flags indicating which modifier keys are down (pressed): Ctrl, Shift, Alt, Win.
Not for UI code (forms, WPF).
IsMod
Returns true if some modifier keys are down (pressed): Ctrl, Shift, Alt, Win.
Not for UI code (forms, WPF).
IsPressed
Returns true if the specified key or mouse button is down (pressed).
Not for UI code (forms, WPF).
Key
Sends virtual keystrokes to the active window. Also can send text, wait, etc.
Send
Sends keys, text and executes other events added with the AddX functions.
Text
Sends text to the active window, using virtual keystrokes or the clipboard. Also can send non-text keystrokes.
WaitForHotkey
Registers a temporary hotkey and waits for it.
WaitForKey(Double, Boolean, Boolean)
Waits for key-down or key-up event of any key, and gets the key code.
WaitForKey(Double, KKey, Boolean, Boolean)
Waits for key-down or key-up event of the specified key.
WaitForKey(Double, String, Boolean, Boolean)
Waits for key-down or key-up event of the specified key.
WaitForNoModifierKeys
Waits while some modifier keys (Ctrl, Shift, Alt, Win) are down (pressed). See Keyb.IsMod.
WaitForNoModifierKeysAndMouseButtons
Waits while some modifier keys (Ctrl, Shift, Alt, Win) or mouse buttons are down (pressed).
WaitForReleased(Double, KKey)
Waits while the specified keys or/and mouse buttons are down (pressed).
WaitForReleased(Double, String)
Waits while the specified keys or/and mouse buttons are down (pressed).
