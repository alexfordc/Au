Remarks
The thread that uses hooks must process Windows messages. For example have a window/dialog/messagebox, or use a 'wait-for' function that dispatches messages or has such option (see Opt.WaitFor).
The variable must be disposed, either explicitly (call Dispose or Unhook) or with the 'using' pattern.
Examples
Copy
bool stop = false;
using(new Au.Util.AccHook(AccEVENT.SYSTEM_FOREGROUND, 0, x =>
{
    Print(x.wnd);
    var a = x.GetAcc();
    Print(a);
    if(x.wnd.ClassNameIs("Shell_TrayWnd")) stop = true;
})) {
    MessageBox.Show("hook");
    //or
    //WaitFor.MessagesAndCondition(-10, () => stop); //wait max 10 s for activated taskbar
    //Print("the end");
}
