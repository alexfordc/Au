ColorIntAdjustLuminance Method

Changes color's luminance (makes darker or brighter).
Returns new color. Does not modify this variable.

public ColorInt AdjustLuminance(
	int n,
	bool totalRange = false
)

Parameters

n
Type: SystemInt32
The luminance in units of 0.1 percent of the range (which depends on totalRange). Can be from -1000 to 1000.
totalRange (Optional)
Type: SystemBoolean
If true, n is in the whole luminance range (from minimal to maximal possible). If false, n is in the range from current luminance of the color to the maximal (if n positive) or minimal (if n negative) luminance.

Return Value

Type: ColorInt
Remarks
Calls API ColorAdjustLuma.
Does not change hue and saturation. Does not use alpha.
