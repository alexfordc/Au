Misc Properties

The WndMisc type exposes the following members.

Properties
  NameDescription
WndActiveOfThisThread
Gets the active window of this thread.
Calls API GetActiveWindow.
WndDesktop
Gets the desktop window.
It displays desktop icons and wallpaper in its child control WndDesktopControl. The "Show Desktop" command (Win+D) activates it.
 Note
It is not the same as API GetDesktopWindow (see WndRoot)
.
 Note
This function is not very reliable. May stop working on a new Windows version or don't work with a custom shell.
WndDesktopControl
Gets the control of "SysListView32" class that contains desktop icons and wallpaper. It is a child of WndDesktop.
 Note
This function is not very reliable. May stop working on a new Windows version or don't work with a custom shell.
WndRoot
Gets the virtual parent window of all top-level windows.
Calls API GetDesktopWindow.
 Note
It is not the visible desktop window (see WndDesktop)
.
WndShell
Gets a window of the shell process (usually process "explorer", class name "Progman").
Calls API GetShellWindow.
 Note
In most cases it is not the window that contains desktop icons (see WndDesktop). But it belongs to the same thread.
WndTop
Gets the very first top-level window in the Z order.
Usually it is a topmost window.
Calls API GetTopWindow(default(Wnd)).
