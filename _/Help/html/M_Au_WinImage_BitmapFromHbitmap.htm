Remarks
How this function is different from FromHbitmap(IntPtr) :
1. Image.FromHbitmap usually creates bottom-up bitmap, which is incompatible with Find(WIArea, Object, WIFlags, Int32, FuncWinImage, Boolean). This function creates normal top-down bitmap, like new Bitmap(...), Bitmap.FromFile(...) etc do.
2. This function always creates bitmap of PixelFormat Format32bppRgb.
