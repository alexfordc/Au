IconsGetFileIconImage Method

Gets file icon.
Extracts icon directly from the file, or gets shell icon, depending on file type, icon index, flags etc.
Calls GetFileIconHandle(String, Int32, GIFlags) and converts to Bitmap. Returns null if failed, for example if the file does not exist.
Later call Dispose().
