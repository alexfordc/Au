NoClass Class

Often-used static functions, to be used like Func instead of Class.Func.
Mostly aliases of functions of this library.
In C# source files add using static Au.NoClass; . Then you can use:
Print instead of Output.Write ;
Empty instead of string.IsNullOrEmpty ;
and more.
