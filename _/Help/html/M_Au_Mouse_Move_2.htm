Exceptions
ExceptionCondition
ArgumentOutOfRangeExceptionThe specified x y is not in screen. No exception if Options.Relaxed is true (then moves to a screen edge).
AuExceptionFailed to move to the specified x y. Some reasons: 1. Another thread blocks or modifies mouse input (API BlockInput, mouse hooks, frequent API SendInput etc); 2. The active window belongs to a process of higher UAC integrity level; 3. Some application called API ClipCursor. No exception if Options.Relaxed is true (then final cursor position is undefined).
Remarks
Uses Options.MouseMoveSpeed, Options.MouseClickSleep (sleeps Math.Min(Options.MouseClickSleep, 7)), Options.Relaxed.
