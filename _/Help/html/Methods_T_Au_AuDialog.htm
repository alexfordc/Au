AuDialog Methods

The AuDialog type exposes the following members.

Methods
  NameDescription
SetButtons
Sets common and/or custom buttons and custom buttons style.
SetCheckbox
Adds check box (if text is not null/empty).
To get check box state after closing the dialog, use the IsChecked property of the DResult variable returned by ShowDialog or the Result property.
SetEditControl
Adds Edit or Combo control (if editType is not DEdit.None (0)).
To get its text after closing the dialog, use the EditText property of the DResult variable returned by ShowDialog or the Result property.
SetExpandControl
Set properties of the control that shows and hides text added by SetExpandedText.
SetExpandedText
Adds text that the user can show and hide.
SetFooterText(String)
Adds text and common icon at the bottom of the dialog.
SetFooterText(String, DIcon)
Adds text and common icon at the bottom of the dialog.
SetFooterText(String, Icon)
Adds text and custom icon at the bottom of the dialog.
SetIcon(Icon)
Sets custom icon.
SetIcon(DIcon)
Sets common icon.
SetOwnerWindow
Sets owner window.
The owner window will be disabled, and this dialog will be on top of it.
This window will be in owner's screen, if screen was not explicitly specified with the Screen property. DefaultScreen is ignored.
SetRadioButtons
Adds radio buttons.
To get selected radio button id after closing the dialog, use the RadioButton property of the DResult variable returned by ShowDialog or the Result property.
SetText
Sets text.
SetTimeout
Let the dialog close itself after closeAfterS seconds.
On timeout ShowDialog returns DResult.Timeout.
Example: d.SetTimeout(30, "OK");
SetTitleBarText
Changes title bar text.
If you don't call this method or title is null or "", dialogs will use DefaultTitle.
SetXY
Sets dialog position in screen.
Show
Shows dialog.
Returns selected button id.
ShowDialog
Shows the dialog.
Returns selected button id and other results packed in a DResult variable.
Call this method after setting text and other properties.
ShowDialogNoWait
Shows the dialog in new thread and returns without waiting until it is closed.
ShowError
Shows dialog with DIcon.Error icon.
Calls Show(String, String, String, DIcon, DFlags, DOwner, String).
ShowEx
Shows dialog.
Returns selected button id and other results packed in a DResult variable.
ShowYesNo
Shows dialog with Yes and No buttons.
Returns true if selected Yes.
Calls Show(String, String, String, DIcon, DFlags, DOwner, String).
ShowInfo
Shows dialog with DIcon.Info icon.
Calls Show(String, String, String, DIcon, DFlags, DOwner, String).
ShowList
Shows dialog with a list of command-link buttons.
Returns 1-based index of the selected button. Returns 0 if clicked the X (close window) button or pressed Esc.
ShowListEx
Shows dialog with a list of command-link buttons.
Returns results packed in a DResult variable. Its Button property is id of the selected button, which is its 1-based index in the list; it is 0 if clicked the X (close window) button or pressed Esc.
The return value can be assigned to an int variable or used in switch; then it is the id (1-based index or 0).
ShowNoWait
Shows dialog like Show(String, String, String, DIcon, DFlags, DOwner, String) but does not wait.
Creates dialog in other thread and returns without waiting until it is closed.
Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send; through the Send property you can modify controls and close the dialog (see example ).
Parameters are the same as with Show(String, String, String, DIcon, DFlags, DOwner, String).
ShowNoWaitEx
Shows dialog like ShowEx(String, String, String, DIcon, DFlags, DOwner, String, String, String, String, String, Int32, Coord, Coord, Int32, ActionDEventArgs) but does not wait.
Creates dialog in other thread and returns without waiting until it is closed.
Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send ; through the Send property you can modify controls and close the dialog (see example).
Parameters are the same as with ShowEx(String, String, String, DIcon, DFlags, DOwner, String, String, String, String, String, Int32, Coord, Coord, Int32, ActionDEventArgs).
ShowNumberInput
Shows dialog with a number edit field and buttons OK and Cancel, and gets that number.
Returns true if selected OK, false if Cancel.
ShowOKCancel
Shows dialog with OK and Cancel buttons.
Returns true if selected OK.
Calls Show(String, String, String, DIcon, DFlags, DOwner, String).
ShowProgress
Shows dialog with progress bar.
Creates dialog in other thread and returns without waiting until it is closed.
Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send ; through the Send property you can set progress, modify controls and close the dialog (see example).
All parameters except marquee are the same as with ShowEx(String, String, String, DIcon, DFlags, DOwner, String, String, String, String, String, Int32, Coord, Coord, Int32, ActionDEventArgs).
ShowProgressEx
Shows dialog with progress bar.
Creates dialog in new thread and returns without waiting until it is closed.
Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send ; through the Send property you can set progress, modify controls and close the dialog (see example).
Most parameters are the same as with ShowEx(String, String, String, DIcon, DFlags, DOwner, String, String, String, String, String, Int32, Coord, Coord, Int32, ActionDEventArgs).
ShowTextInput
Shows dialog with a text edit field and buttons OK and Cancel, and gets that text.
Returns true if selected OK, false if Cancel.
ShowTextInputEx
Shows dialog with a text edit field, buttons OK and Cancel, optionally check box, radio buttons and custom buttons.
Returns results packed in a DResult variable: selected button id (1 for OK, 2 for Cancel), text and check box state.
ShowWarning
Shows dialog with DIcon.Warning icon.
Calls Show(String, String, String, DIcon, DFlags, DOwner, String).
ThreadWaitForClosed
Can be used by other threads to wait until the dialog is closed.
ThreadWaitForOpen
Can be used by other threads to wait until the dialog is open.
If returns true, the dialog is open and you can send messages to it.
If returns false, the dialog is already closed or failed to show.
