AuScriptTriggers Property

Gets or sets an Triggers.ActionTriggers instance, as a field of this class.

Namespace:  Au.Types
Assembly:  Au (in Au.dll) Version: 1.0.0.0
Syntax
C#
Copy
protected ActionTriggers Triggers { get; set; }

Property Value

Type: ActionTriggers
Remarks
This property can be used in automation scripts to avoid creating an ActionTriggers variable explicitly. The returned value is a field of this class. The ActionTriggers object is auto-created when callig this property first time or after setting it = null.
In automation scripts this property is available because this class is the base of the Script class. In other classes need to create an ActionTriggers variable explicitly. In scripts you also can create explicitly if you like, for example to have more than one instance.
Examples
Copy
Triggers.Hotkey["Ctrl+K"] = o => Print(o.Trigger);
Triggers.Run();
