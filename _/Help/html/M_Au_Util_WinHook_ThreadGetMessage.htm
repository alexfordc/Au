WinHookThreadGetMessage Method

Sets a WH_GETMESSAGE hook for a thread of this process.
See API SetWindowsHookEx.

Namespace:  Au.Util
Assembly:  Au (in Au.dll) Version: 1.0.0.0
Syntax
C#
Copy
public static WinHook ThreadGetMessage(
	Action<HookDataThreadGetMessage> hookProc,
	int threadId = 0,
	bool setNow = true
)

Parameters

hookProc
Type: SystemActionHookDataThreadGetMessage
The hook procedure (function that handles hook events).
Must return as soon as possible.
The event cannot be cancelled. As a workaround, you can set msg->message=0. Also can modify other fields.
 Note
When the hook procedure returns, the pointer field of the parameter variable becomes invalid and unsafe to use.
threadId (Optional)
Type: SystemInt32
Native thread id, or 0 for this thread. The thread must belong to this process.
setNow (Optional)
Type: SystemBoolean
Set hook now. Default true.

Return Value

Type: WinHook
Returns a new Util.WinHook object that manages the hook.
