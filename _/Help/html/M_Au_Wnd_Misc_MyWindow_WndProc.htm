WndMiscMyWindowWndProc Method

Calls the window procedure of the window class (see Create(String, String, UInt32, UInt32, Int32, Int32, Int32, Int32, Wnd, LPARAM) ) and manages the lifetime of this variable.
Your derived class should override this function. Your WndProc must call it (base.WndProc) and return its return value, except when don't need default processing. Always call it on WM_NCCREATE and WM_NCDESTROY.
More info: Window Procedures.
