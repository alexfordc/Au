Returns array of substrings delimited by regular expression matches.

Parameters

s
Type: SystemString
Subject string. Cannot be null.
maxCount (Optional)
Type: SystemInt32
The maximal count of substrings to get. The last substring contains the unsplit remainder of the subject string. If 0 (default) or negative, gets all.
more (Optional)
Type: Au.TypesRXMore

Remarks
Element 0 of the returned array is s substring until the first match of the regular expression, element 1 is substring between the first and second match, and so on. If no matches, the array contains single element and it is s .

This function is similar to Split(String, Int32).
Examples
Copy
var s = "one, two,three , four";
var x = new Regex_(@" *, *");
var a = x.Split(s);
