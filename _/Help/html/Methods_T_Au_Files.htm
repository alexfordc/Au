The Files type exposes the following members.

Methods
  NameDescription
CalculateDirectorySize
Calls EnumDirectory(String, FEFlags, FuncFEFile, Boolean, ActionString) and returns sum of all file sizes.
With default flags, it includes sizes of all descendant files, in this directory and all subdirectories except in inaccessible [sub]directories.
Copy
Copies file or directory.
CopyTo
Copies file or directory into another directory.
CreateDirectory
Creates new directory if does not exists.
If need, creates missing parent/ancestor directories.
Returns true if created new directory, false if the directory already exists. Throws exception if fails.
CreateDirectoryFor
Creates parent directory for a new file, if does not exist.
The same as CreateDirectory(String, String), just removes filename from filePath.
Delete(IEnumerableString, Boolean)
Deletes multiple files or/and directories.
The same as Delete(String, Boolean), but faster when using Recycle Bin.
Delete(String, Boolean)
Deletes file or directory.
Does nothing if it does not exist (no exception).
EnumDirectory
Gets names and other info of files and subdirectories in the specified directory.
Returns an enumerable collection of FEFile objects containing the info.
By default gets only direct children. Use flag AndSubdirectories to get all descendants.
ExistsAs
Gets file system entry type - file, directory, and whether it exists.
Returns NotFound (0) if does not exist or if fails to get attributes.
Calls API GetFileAttributes.
ExistsAs2
Gets file system entry type - file, directory, symbolic link, whether it exists and is accessible.
Returns NotFound (0) if does not exist. Returns AccessDenied (< 0) if exists but this process cannot access it and get attributes.
Calls API GetFileAttributes .
The same as ExistsAs(String, Boolean) but provides more complete result. In most cases you can use ExistsAs, it's simpler.
ExistsAsAny
Returns true if file or directory exists.
Calls ExistsAs2(String, Boolean), which calls API GetFileAttributes.
ExistsAsDirectory
Returns true if directory (folder or drive) exists.
Returns false if does not exist or if fails to get its attributes.
Calls ExistsAs(String, Boolean), which calls API GetFileAttributes.
ExistsAsFile
Returns true if file exists and is not a directory.
Returns false if does not exist or if fails to get its attributes.
Calls ExistsAs(String, Boolean), which calls API GetFileAttributes.
GetAttributes
Gets file or directory attributes.
Returns false if the file/directory does not exist.
Calls API GetFileAttributes.
GetProperties
Gets file or directory attributes, size and times.
Returns false if the file/directory does not exist.
Calls API GetFileAttributesEx.
Move
Moves (changes path of) file or directory.
MoveTo
Moves file or directory into another directory.
Rename
Renames file or directory.
SearchPath
Finds file or directory and returns full path.
Returns null if cannot be found.
If the path argument is full path, calls ExistsAsAny(String, Boolean) and returns normalized path if exists, null if not.
Else searches in these places:
1. dirs, if used.
2. ThisApp .
3. Calls API SearchPath, which searches in process directory, Windows system directories, current directory, PATH environment variable. The search order depends on API SetSearchPathMode or registry settings.
4. If path ends with ".exe", tries to get path from registry "App Paths" keys.
