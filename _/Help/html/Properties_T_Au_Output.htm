Output Properties

The Output type exposes the following members.

Properties
  NameDescription
IgnoreConsole
If true, Write and related functions in console process don't not use the console window. Then everything is like in non-console process.
IsConsoleProcess
Returns true if this is a console process.
IsWritingToConsole
Returns true if is writing to console, false if to the output window or log file. Assuming that Writer is not changed.
Does not write to console in these cases: IsConsoleProcess is false. IgnoreConsole is true. LogFile is not null. The startup info of this process tells to not show console window and to not redirect the standard output.
LogFile
Sets log file path.
When set (not null), text passed to Output.Write, Print and similar functions will be written to the file. Assuming that Writer is not changed.
If value is null - restores default behavior.
LogFileTimestamp
Let Write etc also add current time when using log file (see LogFile ).
The time is local, not UTC.
RedirectConsoleOutput
Let Console.WriteX methods in non-console process write to the same destination as Output.Write etc.
RedirectDebugOutput
Let Debug.WriteX and Trace.WriteX methods write to the same destination as Output.Write etc.
Writer
Gets or sets object that actually writes text when is called Output.Write, Print and similar functions.
