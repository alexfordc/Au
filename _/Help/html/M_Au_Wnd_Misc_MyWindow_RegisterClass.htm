Can be used to specify more fields of WNDCLASSEX that is passed to RegisterClassEx .
Defaults: hCursor = arrow; hbrBackground = COLOR_BTNFACE+1; style = CS_GLOBALCLASS; others = 0/null/default.

Remarks
Calls API RegisterClassEx .
The class then can be used in all appdomains of this process. To create windows, use Create(String, String, UInt32, UInt32, Int32, Int32, Int32, Int32, Wnd, LPARAM) .
Each class derived from MyWindow provides its own window procedure (overrides WndProc(Wnd, UInt32, LPARAM, LPARAM)). Windows created not with Create(String, String, UInt32, UInt32, Int32, Int32, Int32, Int32, Wnd, LPARAM) use API DefWindowProc as window procedure, and therefore are not useful.
The window class remains registered until this process ends. Don't need to unregister.
This function can be called multiple times for the same class, for example called once in each appdomain. Next time it just returns class atom.
Thread-safe.
 Note
Don't use code like static ushort _atom = Wnd.Misc.MyWindow.RegisterClass("MyClass");, because in Release configuration compiler removes this code if _atom is not used. Instead you can call this function in a static constructor.
