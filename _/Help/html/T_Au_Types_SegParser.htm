SegParser Structure

Splits a string or StringSegment into StringSegments.
Used with foreach. Also used internally by some functions of this library, for example Split_(String, String, SegFlags) and SplitLines_(String, Boolean) .
Normally you don't create Segmenter instances explicitly; instead use Segments_(String, String, SegFlags) or Split(String, SegFlags) with foreach.
