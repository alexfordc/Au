Remarks
Used like this:





Usually function options are specified using either function parameters or class properties.
Many functions allow to adjust their behavior through some kind of options (aka settings, flags, parameters). There are two common ways to pass options to a function: 1. Function parameters, like Class.Func(other parameters, option1, option2); 2. Class properties, like var x=new Class(); x.Option1=1; x.Option2=2; x.Func(parameters); .
