WinImage Class

Captures, finds and clicks images and colors in windows.
An image is any visible rectangular part of window. A color is any visible pixel of window.
A WinImage variable holds results of Find(WIArea, Object, WIFlags, Int32, FuncWinImage, Boolean) and similar functions (rectangle etc).

Wait

Returns WinImage object containing the rectangle of the found image.

WaitNot
