WaitForWindowActive Method (Double, String, String, String, WFFlags, FuncWnd, Boolean, Object, Boolean)

Waits until window exists and is the active window. Or the opposite, if not is true.
Returns window handle. On timeout returns default(Wnd) if secondsTimeout is negative; else exception.
All undocumented parameters etc are the same as Find(String, String, String, WFFlags, FuncWnd, Boolean, Object) .
If you have a window's Wnd, use WindowActive(Wnd, Double, Boolean) instead.

Remarks
Hidden and minimized windows also can be active. This function by default waits until the window is active and visible. To ignore visibility, use flag HiddenToo . To wait until the window also is not minimized, use the 'also' parameter, like in the example.

More examples: see WindowExists(Double, String, String, String, WFFlags, FuncWnd, Boolean, Object, Boolean).
