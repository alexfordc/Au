KeybWaitForKey Method (Double, Boolean, Boolean)

Waits for key-down or key-up event of any key, and gets the key code.

Namespace:  Au
Assembly:  Au (in Au.dll) Version: 1.0.0.0
Syntax
C#
Copy
public static KKey WaitForKey(
	double secondsTimeout,
	bool up = false,
	bool block = false
)

Parameters

secondsTimeout
Type: SystemDouble
The maximal time to wait, seconds. If 0, waits infinitely. If >0, after that time interval throws System.TimeoutException. If <0, after that time interval returns false.
up (Optional)
Type: SystemBoolean
Wait for key-up event.
block (Optional)
Type: SystemBoolean
Make the event invisible for other apps. If up is true, makes the down event invisible too, if it comes while waiting for the up event.

Return Value

Type: KKey
Returns the key code. On timeout returns 0 if secondsTimeout is negative; else exception.
For modifier keys returns the left or right key code, for example LCtrl/RCtrl, not Ctrl.
Exceptions
ExceptionCondition
TimeoutExceptionsecondsTimeout time has expired (if > 0).
Remarks
Unlike Keyb.WaitForReleased, waits for key event, not for key state.
Uses low-level keyboard hook. Can wait for any single key. See also Keyb.WaitForHotkey.
Ignores key events injected by functions of this library.
Examples
Copy
var key = Keyb.WaitForKey(0, up: true, block: true);
Print(key);
