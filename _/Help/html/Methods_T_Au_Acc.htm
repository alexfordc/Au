Dispose
Releases IAccessible COM object and clears this variable.

DoAction
Performs the object's default action (see DefaultAction ). Usually it is 'click', 'press' or similar.
Uses IAccessible.accDoDefaultAction.
DoActionAndWaitForNewWebPage
Calls DoAction or action and waits until window name changes and web page name changes.
Returns true. On timeout returns false if secondsTimeout is negative; else exception.

Returns the found AO. Returns null if not found.
Parameters etc are as with Find(Wnd, String, String, String, AFFlags, FuncAcc, Boolean, Int32, WndChildFinder).
Find(Wnd, String, String, String, AFFlags, FuncAcc, Boolean, Int32, WndChildFinder)
Finds accessible object (AO) in window.
Returns the found AO. Returns null if not found. See examples.
FindAll(String, String, String, AFFlags, FuncAcc, Boolean)
Finds all matching descendant accessible objects (AO) of this AO.
Returns array of 0 or more elements.
Parameters etc are as with Find(Wnd, String, String, String, AFFlags, FuncAcc, Boolean, Int32, WndChildFinder).

Wait(Double, String, String, String, AFFlags, FuncAcc, Boolean, Int32)
Finds a descendant accessible object (AO) of this AO. Waits until the AO is found or the given time expires.
Returns the found AO. On timeout returns null if secondsTimeout is negative; else exception.
Parameters etc are as with Find(Wnd, String, String, String, AFFlags, FuncAcc, Boolean, Int32, WndChildFinder).

Finds accessible object (AO) in window. Waits until the AO is found or the given time expires.
Returns the found AO. On timeout returns null if secondsTimeout is negative; else exception.
Parameters etc are as with Find(Wnd, String, String, String, AFFlags, FuncAcc, Boolean, Int32, WndChildFinder).
