Remarks
Uses API GetAsyncKeyState.
When processing user input in UI code (forms, WPF), instead use class Keyb.UI or .NET functions. They use API GetKeyState.
Can be used for mouse buttons too, for example Keyb.IsPressed(KKey.MouseLeft). When mouse left and right buttons are swapped, gets logical state, not physical.
