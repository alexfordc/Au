Screen_FromObject Method

Gets Screen object by index or of the screen that contains (or is nearest to) the specified window, rectangle or point.

Namespace:  Au

public static Screen FromObject(
	Object screen

Parameters

screen
Type: SystemObject
Depends on type:
  • null - gets primary screen.
  • Screen - returns it if valid. If invalid, gets primary screen.
  • int - 1-based non-primary screen index (see FromIndex(Int32, Boolean)), or Screen_.Primary (0), Screen_.OfMouse (-1), Screen_.OfActiveWindow (-2).
  • Wnd - gets screen of this window or control (see FromWindow(Wnd)). If invalid, gets primary screen.
  • Control - gets screen of this .NET form or control (see FromControl(Control)). If invalid, gets primary screen.
  • Point - gets screen of this point (see FromPoint(Point)).
  • Rectangle, RECT - gets screen of this rectangle (see FromRectangle(Rectangle)).
  • Acc - gets screen of this accessible object (see FromRectangle(Rectangle)).

Remarks
If something fails (except if invalid index), gets primary screen.
Uses AllScreens to get screen indices. They may not match the indices that you can see in Control Panel.
