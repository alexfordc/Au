text

The action runs when the user types this text and a postfix character or key. By default case-insensitive.
flags (Optional)
Type: SystemNullableTAFlags
Options. If omitted or null, uses Triggers.AutotextTriggers.DefaultFlags. Some flags are used by Triggers.AutotextTriggerArgs.Replace.
postfixType (Optional)
Type: SystemNullableTAPostfix
Postfix type (character, key, any or none). If omitted or null, uses Triggers.AutotextTriggers.DefaultPostfixType; default - a non-word character or the Ctrl key.
postfixChars (Optional)
Type: SystemString
Postfix characters used when postfix type is Char or CharOrKey (default). If omitted or null, uses Triggers.AutotextTriggers.DefaultPostfixChars; default - non-word characters.
