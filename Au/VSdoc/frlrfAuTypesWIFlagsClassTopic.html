WindowDC Get pixels from the device context (DC) of the window client area, not from screen DC. Not used when area is Bitmap. Notes: Usually much faster. Can get pixels from window parts that are covered by other windows or offscreen. But not from hidden and minimized windows. Does not work on Windows 7 if Aero theme is turned off. Then this flag is ignored. If the window is DPI-scaled, the image must be captured from its non-scaled version. Cannot find images in some windows (including Windows Store apps), and in some window parts (glass). All pixels captured from these windows/parts are black.
