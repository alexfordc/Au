Creates named shared memory of specified size. Opens if already exists. Returns shared memory address in this process. Calls API <msdn>CreateFileMapping</msdn> and API <msdn>MapViewOfFile</msdn>.

Remarks
Once the memory is created, it is alive at least until this process ends. Other processes can keep the memory alive even after that. There is no Close function to close the native shared memory object handle. The OS closes it when this process ends.
