TryRun(string, string, SRFlags, SRMore) Calls Run and handles exceptions. All parameters are the same. If Run throws exception, shows warning with its Message and returns int.MinValue. This is useful when you don't care whether Run succeeded, for example in AuMenu menu item command handlers. Using try/catch there would not look good. Handles only exception of type AuException.
