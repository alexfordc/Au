ShowInfo(string, string, string, DFlags, DOwner, string) Shows dialog with DIcon.Info icon. Calls Show.
ShowList(object, string, string, DFlags, DOwner) Shows dialog with a list of command-link buttons. Returns 1-based index of the selected button. Returns 0 if clicked the X (close window) button or pressed Esc.
ShowListEx(object, string, string, DFlags, DOwner, string, string, string, string, int, Coord, Coord, out int, Action<DEventArgs>) Shows dialog with a list of command-link buttons. Returns results packed in a DResult variable. Its Button property is id of the selected button, which is its 1-based index in the list; it is 0 if clicked the X (close window) button or pressed Esc. The return value can be assigned to an int variable or used in switch; then it is the id (1-based index or 0).
ShowNoWait(string, string, string, DIcon, DFlags, DOwner) Shows dialog like Show but does not wait. Creates dialog in other thread and returns without waiting until it is closed. Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send; through the Send property you can modify controls and close the dialog (see example). Parameters are the same as with Show.
ShowNoWaitEx(string, string, string, DIcon, DFlags, DOwner, string, string, string, string, string, int, Coord, Coord, out int, Action<DEventArgs>) Shows dialog like ShowEx but does not wait. Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send; through the Send property you can modify controls and close the dialog (see example). Parameters are the same as with ShowEx.
ShowNumberInput(out int, string, string, DEdit, object, DFlags, DOwner) Shows dialog with a number edit field and buttons OK and Cancel, and gets that number. Returns true if selected OK, false if Cancel.
ShowOKCancel(string, string, DIcon, DFlags, DOwner, string) Shows dialog with OK and Cancel buttons. Returns true if selected OK. Calls Show.
ShowProgress(bool, string, string, string, DFlags, DOwner, Coord, Coord) Shows dialog with progress bar. Creates dialog in other thread and returns without waiting until it is closed. Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send; through the Send property you can set progress, modify controls and close the dialog (see example). All parameters except marquee are the same as with ShowEx.
ShowProgressEx(bool, string, string, string, DFlags, DOwner, string, string, string, string, string, Coord, Coord, out int, Action<DEventArgs>) Shows dialog with progress bar. Creates dialog in new thread and returns without waiting until it is closed. Returns AuDialog variable that can be used to communicate with the dialog using these methods and properties: IsOpen, ThreadWaitForClosed, Result (when closed), DialogWindow, Send; through the Send property you can set progress, modify controls and close the dialog (see example). Most parameters are the same as with ShowEx.
ShowTextInput(out string, string, string, DEdit, object, DFlags, DOwner) Shows dialog with a text edit field and buttons OK and Cancel, and gets that text. Returns true if selected OK, false if Cancel.
ShowTextInputEx(string, string, DEdit, object, DFlags, DOwner, string, string, string, string, string, Coord, Coord, out int, Action<DEventArgs>, string, Action<DEventArgs>) Shows dialog with a text edit field, buttons OK and Cancel, optionally check box, radio buttons and custom buttons. Returns results packed in a DResult variable: selected button id (1 for OK, 2 for Cancel), text and check box state.
ShowWarning(string, string, string, DFlags, DOwner, string) Shows dialog with DIcon.Warning icon. Calls Show.
ShowYesNo(string, string, DIcon, DFlags, DOwner, string) Shows dialog with Yes and No buttons. Returns true if selected Yes. Calls Show.
ThreadWaitForClosed() Can be used by other threads to wait until the dialog is closed.
ThreadWaitForOpen() Can be used by other threads to wait until the dialog is open. If returns true, the dialog is open and you can send messages to it. If returns false, the dialog is already closed or failed to show.
