RXCalloutData Structure

Managed version of PCRE API struct pcre2_callout_block. When you set Callout, your callout function's parameter is of this type. More info in PCRE help topic pcre2callout. Most properties are pcre2_callout_block fields as documented in PCRE help. Other properties and methods are easier/safer versions of unsafe fields like offset_vector.

Result Sets the return value of the callout function, as documented in PCRE help topic pcre2callout. Default 0. If 1, matching fails at the current point, but the testing of other matching possibilities goes ahead, just as if a lookahead assertion had failed. If -1 (PCRE2_ERROR_NOMATCH), the match function returns false (no match). Values less tan -2 are PCRE error codes and cause exception.
