Remarks
A StringSegment variable holds a reference to the string containing the substring (Buffer), the start index of the substring (Offset) and substring length (Length). A new string object is allocated only by functions that return a string, eg Value. One of ways to create StringSegment instances is to split a string with Segments_(string,string,SegFlags). Functions throw ArgumentOutOfRangeException, ArgumentNullException or NullReferenceException exception when an argument is invalid. The indexer throws IndexOutOfRangeException. Don't use the default constructor. Then Buffer is null and the behavior of functions is undefined. Other constructors throw exception if the string is null.
