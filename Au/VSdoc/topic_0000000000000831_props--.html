Wnd Properties

Wnd Structure  See Also

The following tables list the members exposed by the Wnd type.

Properties
Name Description
AsButton Casts this to WButton.
ClassName Gets class name. Returns null if fails, eg if the window is closed. Supports GetError.
ClientHeight Gets client area height.
ClientRect Gets client area rectangle (width and height). The same as ClientSize, just the return type is different.
ClientRectInScreen Gets client area rectangle in screen coordinates.
ClientSize Gets client area width and height. The same as ClientRect, just the return type is different.
ClientWidth Gets client area width.
ControlId Gets or sets id of this control. The 'get' function supports GetError.
ControlText Gets control text. Returns "" if no text. Returns null if fails, eg if the window is closed. Supports GetError.
ExStyle Gets window extended style.
Handle Gets window handle as IntPtr. Code w.Handle is the same as (IntPtr)w .
Height Gets height.
Is0 Returns true if the handle is 0.
Is64Bit Returns true if the window is of a 64-bit process, false if of a 32-bit process. Also returns false if fails. Supports GetError. If Is64BitOS is true, calls API <msdn>GetWindowThreadProcessId</msdn>, <msdn>OpenProcess</msdn> and <msdn>IsWow64Process</msdn>. <note>If you know that the window belongs to current process, instead use Is64BitProcess or IntPtr.Size==8. This function is much slower.</note>
IsActive Returns true if this window is the active (foreground) window.
IsAlive Returns true if the handle identifies an existing window. Returns false if the handle is 0 or invalid. Invalid non-0 handle usually means that the window is closed/destroyed.
IsChildWindow Returns true if this is a child window (control), false if top-level window.
IsCloaked Returns true if the window is cloaked. Returns false if not cloaked or if failed. On Windows 7 returns false because there is no "cloaked window" feature. Windows 10 uses window cloaking mostly to hide windows on inactive desktops. Windows 8 - mostly to hide Metro app windows.
IsCloakedGetState Gets the cloaked state. Returns 0 if not cloaked or if failed. Else returns flags: 1 cloaked by its application, 2 cloaked by Windows, 4 cloaked because its owner window is cloaked. On Windows 7 returns 0 because there is no "cloaked window" feature.
IsConsole Returns true if this is a console window (class name "ConsoleWindowClass").
IsEnabled Gets the enabled state. Returns true if the window is enabled. Returns false if is disabled or is a child of disabled parent. Also returns false when fails (probably window closed or 0 handle). Supports GetError. Calls API <msdn>IsWindowEnabled</msdn>.
IsFocused Returns true if this is the control or window that has the keyboard input focus. Can belong to any process/thread. With controls of this thread you can use the more lightweight function IsFocusedLocal. Calls WndFocused.
IsFocusedLocal Returns true if this is the control or window of this thread that has the keyboard input focus. Calls API <msdn>GetFocus</msdn>.
IsFullScreen Returns true if this is a full-screen window and not desktop.
IsHung Returns true if thread of this window is considered hung (not responding). Calls API <msdn>IsHungAppWindow</msdn>.
IsHungGhost Returns true if the window is a ghost window that the system creates over a hung (not responding) window to allow the user to minimally interact with it.
IsMaximized Returns true if maximized, false if not. Also returns false when fails (probably window closed or 0 handle). Supports GetError. Calls API <msdn>IsZoomed</msdn>.
IsMinimized Returns true if minimized, false if not. Also returns false when fails (probably window closed or 0 handle). Supports GetError. Calls API <msdn>IsIconic</msdn>.
IsOfThisProcess Returns true if this window belongs to the current process, false if to another process. Also returns false when fails (probably window closed or 0 handle). Supports GetError. Calls API <msdn>GetWindowThreadProcessId</msdn>.
IsOfThisThread Returns true if this window belongs to the current thread, false if to another thread. Also returns false when fails (probably window closed or 0 handle). Supports GetError. Calls API <msdn>GetWindowThreadProcessId</msdn>.
IsPopupWindow Returns true if has Native.WS_POPUP style.
IsResizable Returns true if has Native.WS_THICKFRAME style.
IsToolWindow Returns true if has Native.WS_EX_TOOLWINDOW style.
IsTopmost Returns true if this is a topmost (always-on-top) window.
IsUacAccessDenied Returns true if UAC would not allow to automate the window. It happens when current process has lower UAC integrity level and is not uiAccess, unless UAC is turned off.
IsUnicode Returns true if the window is a Unicode window, false if ANSI. Also returns false when fails (probably window closed or 0 handle). Supports GetError. Calls API <msdn>IsWindowUnicode</msdn>.
IsVisible Returns true if the window is visible. Returns false if is invisible or is a child of invisible parent. Also returns false when fails (probably window closed or 0 handle). Supports GetError.
IsVisibleAndNotCloaked Returns true if IsVisible returns true and IsCloaked returns false.
IsVisibleEx Returns true if the window is visible. Returns false if is invisible or is a child of invisible parent. Also returns false when fails (probably window closed or 0 handle). Supports GetError.
IsWindows10StoreApp Returns non-zero if this window is a Windows 10 Store app window: 1 if class name is "ApplicationFrameWindow", 2 if "Windows.UI.Core.CoreWindow".
IsWindows8MetroStyle Returns true if this window has Metro style, ie is not a classic desktop window. On Windows 8/8.1 most Windows Store app windows and many shell windows have Metro style. On Windows 10 few windows have Metro style. On Windows 7 there are no Metro style windows.
LastFind Gets arguments and result of this thread's last call to Find or FindAll.
MouseClientXY Returns mouse pointer position relative to the client area of this window.
Name Gets window or control name. Returns "" if no name. Returns null if fails, eg if the window is closed. Supports GetError.
NameAcc Gets Name of the accessible object (role WINDOW) of this window or control. Returns "" if the object has no name or failed to get it. Returns null if invalid window handle.
NameWinForms Gets Control.Name property of a .NET Windows Forms control. Returns null if it is not a Windows Forms control or if fails. <note>Use this with controls of other processes. Don't use with your controls, when you have a Control object.</note> <note>This is slow when getting names of multiple controls in a window. Instead create a WinFormsControlNames instance and call its GetControlName method for each control.</note>
ProcessId Calls API <msdn>GetWindowThreadProcessId</msdn> and returns process id. Returns 0 if fails. Supports GetError.
ProcessName Gets filename (without ".exe") of process executable file. Return null if fails. Calls ProcessId and Process_.GetProcessName.
ProcessPath Gets full path of process executable file. Return null if fails. Calls ProcessId and Process_.GetProcessName.
Prop Returns an object that manages window properties using API <msdn>SetProp</msdn> and co.
Rect Gets rectangle (position and size) in screen coordinates.
RectInParent Gets or sets child window rectangle in parent window's client area.
Screen Gets Screen object of the screen that contains this window (the biggest part of it) or is nearest to it. If this window handle is default(Wnd) or invalid, gets the primary screen. Calls FromWindow.
Size Gets width and height.
Style Gets window style.
ThreadId Calls API <msdn>GetWindowThreadProcessId</msdn> and returns thread id. Returns 0 if fails. Supports GetError. <note>It is native thread id, not Thread.ManagedThreadId.</note>
Width Gets width.
WndActive Gets the active (foreground) window. Calls API <msdn>GetForegroundWindow</msdn>. Returns default(Wnd) if there is no active window; more info: WaitForAnActiveWindow.
WndDirectParent Gets the window or control that is the direct parent of this control. Returns default(Wnd) if this is a top-level window. If you need the top-level parent window, use WndWindow instead.
WndDirectParentOrOwner Gets the window or control that is the direct parent of this control or owner of this top-level window. Calls API <msdn>GetParent</msdn>. Faster than WndDirectParent.
WndFirstChild Gets the child control at the top of the Z order. Returns default(Wnd) if no children. The same as WndChild(0). Calls API <msdn>GetWindow</msdn>(this, GW_CHILD).
WndFirstSibling Gets the window of the same type that is highest in the Z order. If this is a top-level window, gets first top-level window, else gets first control of the same direct parent. If this is the first, returns this, not default(Wnd). Calls API <msdn>GetWindow</msdn>(this, GW_HWNDFIRST).
WndFocused Gets the control or window that has the keyboard input focus. It can belong to any process/thread. With controls of this thread you can use the more lightweight function WndFocusedLocal. Calls API <msdn>GetGUIThreadInfo</msdn>.
WndFocusedLocal Gets the control or window of this thread that has the keyboard input focus. Calls API <msdn>GetFocus</msdn>.
WndLastChild Gets the child control at the bottom of the Z order. Returns default(Wnd) if no children.
WndLastSibling Gets the window of the same type that is lowest in the Z order. If this is a top-level window, gets last top-level window, else gets last control of the same direct parent. If this is the last, returns this, not default(Wnd). Calls API <msdn>GetWindow</msdn>(this, GW_HWNDLAST).
WndNext Gets the window of the same type that is next (below this) in the Z order. If this is a top-level window, gets next top-level window, else gets next control of the same direct parent. If this is the last, returns default(Wnd). Calls API <msdn>GetWindow</msdn>(this, GW_HWNDNEXT).
WndOwner Gets or sets the owner window of this top-level window.
WndPrev Gets the window of the same type that is previous (above this) in the Z order. If this is a top-level window, gets previous top-level window, else gets previous control of the same direct parent. If this is the first, returns default(Wnd). Calls API <msdn>GetWindow</msdn>(this, GW_HWNDPREV).
WndWindow Gets the top-level parent window of this control. If this is a top-level window, returns this. Returns default(Wnd) if this window is invalid.
X Gets horizontal position in screen coordinates.
Y Gets vertical position in screen coordinates.
