Remarks
If full match, returns true, and result contains the match and all groups that exist in the regular expressions. If partial match, returns true, and result contains the match without groups. Partial match is possible if used a PARTIAL_ flag. If no match, returns false, and result normally is null. But if a mark is available, result is an object with two valid properties - Exists (false) and Mark; other properties have undefined values or throw exception. This function is similar to Match(string).
