InterDomainVariables.GetVariable<T> (String, Func<T>) Method

Example See Also

Gets the value of an inter-domain variable. If the variable does not exist or is null, calls a callback function that provides an initial value; then sets the variable = the initial value. Thread-safe.

Remarks
Be careful with types derived from MarshalByRefObject. If the object does not exist, your callback function creates it in this appdomain. Maybe this appdomain is not where you want it to live. Instead use SetVariable and another overload of GetVariable.
