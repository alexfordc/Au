WaitFor.WindowExists (Double, String, String, String, WFFlags, Func<WndBoolean>, Object, Boolean) Method

Example See Also

Waits until window exists and (optionally) is visible. Or the opposite, if not is true. Returns window handle. On timeout returns default(Wnd) if secondsTimeout is negative; else exception. All undocumented parameters etc are the same as Find.

Parameters

secondsTimeout
Type: double

The maximal time to wait, seconds. If 0, waits indefinitely. If >0, after that time interval throws TimeoutException. If <0, after that time interval returns default(Wnd).

Do the opposite - wait until no such windows exist. The return value is opposite too. If succeeded - default(Wnd). If timeout - a matching window (if no exception). If you have a window's Wnd variable, use WindowClosed or WindowVisible instead.

Type: Wnd

Remarks
By default ignores invisible windows. Use flag HiddenToo if need.

Using in a Form/Control event handler.
Copy 
 var f = new Form();
 f.Click += async (unu, sed) =>
   {
 	  Print("waiting for Notepad...");
 	  Wnd w = await Task.Run(() => WaitFor.WindowExists(-10, "* Notepad"));
 	  if(w.Is0) Print("timeout"); else Print(w);
   };
 f.ShowDialog();
