Remarks
All functions have a secondsTimeout parameter. It is the maximal time to wait, seconds. If it is 0, waits indefinitely. If >0, after that time interval throws TimeoutException. If <0, after that time interval stops waiting and returns default value (false, default(Wnd), etc). While waiting, messages and events are not processed. For example, if used in a Form/Control event handler, the form would stop responding. Then need to use another thread, for example async/await/Task, like in the example.
Example
Copy 
 Wnd w = WaitFor.WindowExists(10, "* Notepad");
 Print(w);
Using in a Form/Control event handler.
Copy 
 var f = new Form();
 f.Click += async (unu, sed) =>
   {
 	  Print("waiting for Notepad...");
 	  Wnd w = await Task.Run(() => WaitFor.WindowExists(-10, "* Notepad"));
 	  if(w.Is0) Print("timeout"); else Print(w);
   };
 f.ShowDialog();
