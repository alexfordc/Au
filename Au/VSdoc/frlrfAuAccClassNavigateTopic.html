Parameters

navig
Type: string

String consisting of one or more navigation direction strings separated by space, like "parent next child4 first".

Can be only 2 letters, like "pr" for "previous". A string like "next3" or "next,3" is the same as "next next next". Except for "child". For "next", "previous", "firstchild", "lastchild", "#N" is used <msdn>IAccessible.accNavigate</msdn>. Not all AOs support it. Some AOs skip invisible siblings. Instead you can use "parent childN" or "childN". For "parent" is used <msdn>IAccessible.get_accParent</msdn>. Few AOs don't support. Some AOs return a different parent than in the AO tree. For "child" is used API <msdn>AccessibleChildren</msdn>.

secondsToWait
Type: double

Wait for the wanted AO max this number of seconds. If negative, waits forever.
