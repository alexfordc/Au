If not 0, auto-close the dialog after this time (seconds) and set result's Button property = Timeout.

A link-clicked event handler function, eg lambda. Enables hyperlinks in small-font text. Example:

Remarks
The returned DResult variable has these properties: selected button id, selected radio button id, check box state. Tip: DResult supports implicit cast to int. You can use code switch(AuDialog.ShowEx(...)) instead of switch(AuDialog.ShowEx(...).Button) . Tip: For optional parameters use named arguments. Example: AuDialog.ShowEx("Text.", icon: DIcon.Info, title: "Title") . This function allows you to use most of the dialog features, but not all. Alternatively you can create a AuDialog class instance, set properties and call ShowDialog. Example in AuDialog class help.
Example
Copy 
 var r = AuDialog.ShowEx("Main text", "More text.", "1 OK|2 Cancel", expandedText: "Expanded text", radioButtons: "1 One|2 Two|3 Three", checkBox: "Check", secondsTimeout: 30);
 Print(r);
 switch(r) {
 case 1: Print("OK"); break;
 case DResult.Timeout: Print("timeout"); break;
 default: Print("Cancel"); break;
 }
