Remarks
Has all the same capabilities as Wildex, but is just a struct of single pointer size. The pointer is Object that can be of one of these types: Wildex - if the assigned string was with wildcard expression options, like "**c text". To compare it with other strings, Match calls Match. String - if the assigned string was without wildcard expression options. To compare it with other strings, Match calls Like_(string,string,bool) with ignoreCase true. null - if was not assigned a non-null string.
