Dispose() Releases IAccessible COM object and clears this variable.
DoAction() Performs the object's default action (see DefaultAction). Usually it is 'click', 'press' or similar. Uses <msdn>IAccessible.accDoDefaultAction</msdn>.
DoActionAndWaitForNewWebPage(out double, Action<Acc>) Calls DoAction or action and waits until window name changes and web page name changes. Returns true. On timeout returns false if secondsTimeout is negative; else exception.
DoJavaAction(string) Performs one of actions supported by this Java accessible object.
Equals(Object) Determines whether the specified object is equal to the current object. (inherited from Object).
Finalize Allows an object to try to free resources and perform other cleanup operations before it is reclaimed by garbage collection. (inherited from Object).
Find(string, string, string, AFFlags, Func<AccBoolean>, int) Finds a descendant accessible object (AO) of this AO. Returns the found AO. Returns null if not found. Parameters etc are as with Find(Wnd,string,string,string,AFFlags,Func<Acc, bool>, int, ChildFinder).
Find(Wnd, string, string, string, AFFlags, Func<AccBoolean>, int, ChildFinder) Finds accessible object (AO) in window. Returns the found AO. Returns null if not found. See examples.
FindAll(string, string, string, AFFlags, Func<AccBoolean>) Finds all matching descendant accessible objects (AO) of this AO. Returns array of 0 or more elements. Parameters etc are as with Find(Wnd,string,string,string,AFFlags,Func<Acc, bool>, int, ChildFinder).
FindAll(Wnd, string, string, string, AFFlags, Func<AccBoolean>) Finds all matching accessible objects in window. Returns array of 0 or more elements. Parameters etc are as with Find(Wnd,string,string,string,AFFlags,Func<Acc, bool>, int, ChildFinder).
Focus(bool) Makes this object focused for keyboard input.
Focused(bool) Gets the accessible object that has the keyboard focus. Returns null if fails.
FromEvent(Wnd, int, int) Gets the accessible object that generated the event that is currently being processed by the callback function used with API <msdn>SetWinEventHook</msdn>. Returns null if failed. Suports GetError.
FromMouse(AXYFlags) Gets accessible object from mouse cursor (pointer) position. Uses API <msdn>AccessibleObjectFromPoint</msdn>.
FromWindow(Wnd, AccOBJID, AWFlags) Gets accessible object of window or control or its standard part - client area, titlebar etc. Uses API <msdn>AccessibleObjectFromWindow</msdn>.
FromXY(Coord, Coord, CoordOptions, AXYFlags) Gets accessible object from point. Uses API <msdn>AccessibleObjectFromPoint</msdn>.
FromXY(Point, AXYFlags) Gets accessible object from point. Uses API <msdn>AccessibleObjectFromPoint</msdn>.
GetHashCode Serves as the default hash function. (inherited from Object).
GetProperties(string, out AccProperties) Gets multiple properties.
GetRect(out RECT) Gets location of this accessible object in screen. Uses <msdn>IAccessible.accLocation</msdn>.
GetRect(out RECT, Wnd) Gets location of this accessible object in the client area of window w. Uses <msdn>IAccessible.accLocation</msdn> and MapScreenToClient(refRECT).
GetType Gets the Type of the current instance. (inherited from Object).
Html(bool) Gets HTML.
HtmlAttribute(string) Gets a HTML attribute.
HtmlAttributes() Gets all HTML attributes.
MemberwiseClone Creates a shallow copy of the current Object. (inherited from Object).
Navigate(string, double) Gets an adjacent or related accessible object (AO) - next, child, parent, etc. Returns null if not found.
PrintAll(Wnd, string, AFFlags, string) Displays properties of all found accessible objects of window w.
ScrollTo() Scrolls this accessible object into view.
Select(AccSELFLAG) Selects or deselects.
ToString() Formats string from main properties of this accessible object.
VirtualClick() Posts mouse-left-click message to the container window, using coordinates of this object.
VirtualRightClick() Posts mouse-right-click message to the container window, using coordinates of this object.
Wait(double, string, string, string, AFFlags, Func<AccBoolean>, int) Finds a descendant accessible object (AO) of this AO. Waits until the AO is found or the given time expires. Returns the found AO. On timeout returns null if secondsTimeout is negative; else exception. Parameters etc are as with Find(Wnd,string,string,string,AFFlags,Func<Acc, bool>, int, ChildFinder).
Wait(double, Wnd, string, string, string, AFFlags, Func<AccBoolean>, int) Finds accessible object (AO) in window. Waits until the AO is found or the given time expires. Returns the found AO. On timeout returns null if secondsTimeout is negative; else exception. Parameters etc are as with Find(Wnd,string,string,string,AFFlags,Func<Acc, bool>, int, ChildFinder).

MouseMove(this Wnd, Coord, Coord, bool) Moves the cursor (mouse pointer) to the position x y relative to this window. Calls Move(Wnd,Coord,Coord,bool). By default x y coordinates are relative to the client area; default - center.
