First() Stores current time in the first element of an internal array.

First(int, params Action[]) Calls SpinCPU(spinCpuMS, codes) and First.
NW(char) Calls Next and Write.
Next(char) Stores current time in next element of an internal array. Don't call Next() more than 16 times after First(), because the array has fixed size.
SpinCPU(int, params Action[]) Repeatedly executes codes (zero or more lambda functions) for the specified number of milliseconds.
Write() Formats a string from time values collected by calling First() and Next(), and shows it in the output. The string contains the number of microseconds of each code execution between calling First() and each Next().
