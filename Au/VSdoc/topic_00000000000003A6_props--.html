The following tables list the members exposed by the AuDialog type.

Properties
Name Description
DefaultButton Specifies which button responds to the Enter key. If 0 or not set, auto-selects.
DialogWindow Gets dialog window handle as Wnd. Returns default(Wnd) if the dialog is not open.
EditControl Gets edit control handle as Wnd.
FlagCanBeMinimized Add 'Minimize' button to the title bar.
FlagEndThread Call Abort() if selected OK button when there are no other buttons. Also when selected Cancel, No, and on timeout.
FlagRtlLayout Right-to left layout. Default is AuDialog.Options.RtlLayout.
FlagShowMarqueeProgressBar Show progress bar that just plays an animation but does not indicate which part of the work is already done.
FlagShowProgressBar Show progress bar.
FlagTopmost Makes the dialog window topmost or non-topmost. If true, will set topmost style when creating the dialog. If false, will not set. If null (default), the dialog will be topmost if both these are true: no owner window, AuDialog.Options.TopmostIfNoOwnerWindow is true.
FlagXCancel Allow to cancel even if there is no Cancel button. It adds X (Close) button to the title bar, and also allows to close the dialog with the Esc key. When the dialog is closed with the X button or Esc, the returned result button id is 0 if there is no Cancel button; else the same as when clicked the Cancel button. The same as XCancel.
IsOpen Returns true if the dialog is open and your code can send messages to it.
Result Gets selected button id and other results packed in a DResult variable. It is the same variable as the ShowDialog return value. If the result is still unavailable (eg the dialog still not closed): If called from the same thread that called ShowDialog, returns null. If called from another thread, waits until the dialog is closed and the return value is available. Note that ShowDialogNoWait calls ShowDialog in another thread.
Screen Sets the screen (display monitor) where to show the dialog in multi-screen environment. If null or not set, will be used owner window's screen or DefaultScreen. If screen index is invalid, the 'show' method shows warning, no exception. More info: FromObject, MoveInScreen.
Send Allows to modify dialog controls while it is open, and close the dialog. Example: d.Send.Close(); . Example: d.Send.ChangeText2("new text", false); . Example: d.Send.Message(TDApi.TDM.CLICK_VERIFICATION, 1); .
Width Sets the width of the dialog's client area. The actual width will depend on DPI (the Windows "text size" setting). If less than default width, will be used default width.
