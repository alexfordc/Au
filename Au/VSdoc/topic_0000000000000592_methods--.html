GetUrlProtocolLength(string) Gets the length of the URL protocol name (also known as URI scheme) in string, including ':'. If the string does not start with a protocol name, returns 0. URL examples: "http:" (returns 5), "http://www.x.com" (returns 5), "file:///path" (returns 5), "shell:etc" (returns 6). The protocol can be unknown, the function just checks string format, which is an ASCII alpha character followed by one or more ASCII alpha-numeric, '.', '-', '+' characters, followed by ':' character.
IsFullPath(string) Returns true if path matches one of these wildcard patterns: @"?:\*" - local path, like @"C:\a\b.txt". Here ? is A-Z, a-z. @"?:" - drive name, like @"C:". Here ? is A-Z, a-z. Note: it is considered full path in this library, but for .NET and Windows API it is not full path. @"\\*" - network path, like @"\\server\share\...". Or has prefix @"\\?\".
IsFullPathExpandEnvVar(ref string) Expands environment variables and calls IsFullPath. Returns true if path matches one of these wildcard patterns: @"?:\*" - local path, like @"C:\a\b.txt". Normalize(string, string, PNFlags) Makes normal full path from path that can contain special substrings etc. The sequence of actions: 1. If path starts with '%' character, expands environment variables (see ExpandEnvVar). 2. If path is not full path but looks like URL, and used flag CanBeUrl, returns path. 3. If path is not full path, and defaultParentDirectory is not null/"", combines path with ExpandEnvVar(defaultParentDirectory). 4. If path is not full path, throws exception. 5. Calls API <msdn>GetFullPathName</msdn>. It replaces '/' to '\\', replaces multiple '\\' to single (where need), processes @"\.." etc, trims spaces, etc. 6. If no flag DoNotExpandDosPath, if contains '~' character, calls API <msdn>GetLongPathName</msdn>. It converts short DOS path to normal path, if possible, for example @"c:\progra~1" to @"c:\program files". It is slow. It converts path only if the file exists. 7. If no flag DoNotRemoveEndSeparator, removes '\\' character at the end, unless it is a drive path (eg @"C:\"). 8. Appends '\\' character if ends with a drive name (eg "C:" -> @"C:\"). 9. If no flag DoNotPrefixLongPath, calls PrefixLongPathIfNeed, which adds @"\\?\" etc prefix if path is very long.
PrefixLongPath(string) If path is full path (see IsFullPath) and does not start with @"\\?\", prepends @"\\?\". If path is network path (like @"\\computer\folder\..."), makes like @"\\?\UNC\computer\folder\...".
PrefixLongPathIfNeed(string) Calls PrefixLongPath if path is longer than MaxDirectoryPathLength (247).
ToString Returns a string that represents the current object. (inherited from Object).
UnprefixLongPath(string) If path starts with @"\\?\" prefix, removes it. If path starts with @"\\?\UNC\" prefix, removes @"?\UNC\".
