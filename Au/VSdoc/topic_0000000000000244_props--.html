IsWin Returns true if Win key is pressed (left or right). See also: ModifierKeys (gets Ctrl, Shift and Alt).
SleepAfter Wait milliseconds when all keys have been sent (before the function returns). Default: 50. Valid values: 0 - 5000 (5 seconds).
SleepIn Wait milliseconds after sending each key down and up event. Default: 0. Valid values: 0 - 1000 (1 second).
