AuDialog.SetEditControl Method

Adds Edit or Combo control (if editType is not DEdit.None (0)). To get its text after closing the dialog, use the EditText property of the DResult variable returned by ShowDialog or the Result property.

Initial text. If editType is DEdit.Combo, it can be a string array, List or IEnumerable; the first item sets combo-box editable text, other items - combo box drop-down list items.

Remarks
The API TaskDialogIndirect does not have an option to add an edit control. This class itself creates it. Does not support progress bar.
