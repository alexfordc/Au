NativeHeap.ReAlloc Method

See Also

Reallocates a memory block to make it bigger or smaller. Returns its address, which in most cases is different than the old memory block address. Preserves data in Math.Min(old size, new size) bytes of old memory (copies from old memory if need). Calls API <msdn>HeapReAlloc</msdn> or <msdn>HeapAlloc</msdn>.

Parameters

mem
Type: void

Old memory address. If null, allocates new memory like Alloc.

size
Type: LPARAM

New byte count.

zeroInit
Type: bool

When size is growing, set all added bytes = 0. If false (default), the added memory is uninitialized, ie random byte values. Slower when true.

Exceptions
Exception type Condition

OutOfMemoryException

Failed. Probably size is too big.

Remarks
The memory is unmanaged and will not be freed automatically. Always call Free when done or ReAlloc if need to resize it without losing data.
