WaitFor.WindowCondition Method

Example See Also

Waits for an user-defined condition of an existing window. Returns true. On timeout returns false if secondsTimeout is negative; else exception.

Parameters

w
Type: Wnd

An existing window. It is passed to the callback function.

condition
Type: Func<Wnd, bool>

Callback function (eg lambda). It is called repeatedly, until returns true.

secondsTimeout
Type: double

The maximal time to wait, seconds. If 0, waits indefinitely. If >0, after that time interval throws TimeoutException. If <0, after that time interval returns false.

doNotThrowIfClosed
Type: bool

Do not throw exception if w is invalid or the window was closed while waiting. In such case the callback function must return false. If need, it can use IsAlive, like in the example. Else the exception is thrown (with a small delay) to prevent infinite waiting.

Return Value

Type: bool
Exceptions
Exception type Condition

TimeoutException

secondsTimeout time has expired.

WndException

w is invalid or the window was closed while waiting.

Example
Copy 
 Wnd w = Wnd.Find("* Notepad");
 
 //wait max 30 s until window w is active. Exception on timeout or if closed.
 WaitFor.WindowCondition(w, t => t.IsActive); //the same as WaitFor.WindowActive(w, 30);
 Print("active");
 
 //wait indefinitely until window w is minimized or closed
 WaitFor.WindowCondition(w, t => t.IsMinimized || !t.IsAlive, 0, true);
 if(!w.IsAlive) { Print("closed"); return; }
 Print("minimized");
