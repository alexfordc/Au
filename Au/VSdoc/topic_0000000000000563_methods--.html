GetAppIconHandle(int) Gets the first native icon handle of the entry assembly of this appdomain. Returns default(IntPtr) if there are no icons. It is not an icon from managed resources. The icon is cached and protected from destroying, therefore don't need to destroy it, and not error to do it.
GetFileIconHandle(string, int, GIFlags) Gets file icon. Extracts icon directly from the file, or gets shell icon, depending on file type, icon index, flags etc. Returns native icon handle. Returns default(IntPtr) if failed, for example if the file does not exist. Later call DestroyIconHandle or HandleToImage. Or instead use GetFileIconImage.
GetFileIconHandleRaw(string, int, int) Extracts icon directly from file that contains it. Returns icon handle. Returns default(IntPtr) if failed. Later call DestroyIconHandle or HandleToImage.
GetFileIconImage(string, int, GIFlags) Gets file icon. Extracts icon directly from the file, or gets shell icon, depending on file type, icon index, flags etc. Calls GetFileIconHandle and converts to Bitmap. Returns null if failed, for example if the file does not exist. Later call Dispose().
GetHashCode Serves as the default hash function. (inherited from Object).
GetPidlIconHandle(Pidl, int) Gets icon of a file or other shell object specified as ITEMIDLIST. Returns native icon handle. Returns default(IntPtr) if failed. Later call DestroyIconHandle or HandleToImage. Or instead use GetPidlIconImage.
GetPidlIconImage(Pidl, int) Gets icon of a file or other shell object specified as ITEMIDLIST. Calls GetPidlIconHandle and converts to Bitmap. Returns null if failed. Later call Dispose.
GetProcessExeIconHandle(int) Gets the first native icon handle of the program file of this process. Returns default(IntPtr) if there are no icons. It is not an icon from managed resources. The icon is cached and protected from destroying, therefore don't need to destroy it, and not error to do it.
GetShellIconSize(ShellSize) Gets the width and height of shell icons of standard sizes - small, large, extra large and jumbo. Jumbo is always 256. Others depend on text size (DPI) that can be changed in Control Panel. If text size is 100%, they usually are 16, 32, 48.
GetShellStockIconHandle(SHSTOCKICONID, int) Gets a shell stock icon handle. Later call DestroyIconHandle or HandleToImage.
