Exceptions
Exception type Condition

AuException

Failed to move to the last x y. Some reasons: 1. Another thread blocks or modifies mouse input (API BlockInput, mouse hooks, frequent API SendInput etc); 2. The active window belongs to a process of higher UAC integrity level; 3. Some application called API ClipCursor. No exception if Options.Relaxed is true (then final cursor position is undefined).

ArgumentOutOfRangeException

The last x y is not in screen. No exception if Options.Relaxed is true (then moves to a screen edge).

ArgumentException

The string is not compatible with this library version (recorded with a newer version and has additional options).

Remarks
Uses Options.MouseClickSleep (sleeps Math.Min(Options.MouseClickSleep, 7) after each movement, except if recorded with sleep times), Options.Relaxed (for the last movement; does not care about intermediate movements). Does not use Options.MouseMoveSpeed.
