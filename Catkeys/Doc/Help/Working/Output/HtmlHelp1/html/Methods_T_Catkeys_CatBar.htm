Add(String, ActionCatMenu_CatBar_BaseClickEventData, Object)
Adds new button as ToolStripButton. Sets its text, icon and Click event handler delegate. Other properties can be specified later. See example. Code t.Add("text", o => Print(o)); is the same as t["text"] = o => Print(o);.
Add(ToolStripItem, Object, ActionCatMenu_CatBar_BaseClickEventData)
Adds item of any supported type, for example ToolStripLabel, ToolStripTextBox, ToolStripComboBox, ToolStripProgressBar. Supports types derived from ToolStripItem.
